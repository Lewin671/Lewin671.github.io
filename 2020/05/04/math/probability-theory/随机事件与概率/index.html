随机事件与概率

Posted on 2020-05-04 Edited on 2020-12-16 In Math , 概率论

随机事件及其运算

样本空间

随机试验是概率论的基本概念，试验的结果事先不能准确地预言，但具有如下三个特征：

(1) 可以在相同的条件下重复进行。

(2) 每次试验的结果不止一个，但是预先知道试验的所有可能结果。

(3) 每次试验前不能确定哪个结果会出现。

随机试验的可能结果称为样本点或基本事件，记为ω\omegaω。 样本点的全体称为样本空间，记为Ω\OmegaΩ，即Ω={ω}\Omega = \{\omega\}Ω={ω}.

例如，下面给出随机现象的样本空间：

  1. 抛一枚硬币的样本空间为Ω1={ω1,ω2}\Omega_{1} = \{\omega_{1},\omega_{2}\}Ω1​={ω1​,ω2​}，其中ω1\omega_{1}ω1​表示正面朝上，ω2\omega_{2}ω2​表示反面朝上。
  2. 一天内进入商场的顾客数的样本空间记为Ω2={0,1,2,⋯ ,105,⋯ }\Omega_{2} = \{0,1,2,\cdots,10^{5},\cdots\}Ω2​={0,1,2,⋯,105,⋯}。

需要注意的是：

  1. 样本空间中的元素可以是数，也可以不是数。
  2. 样本空间至少有两个样本点，仅含有两个样本点的样本空间是最简单的样本空间。

随机事件

随机现象的某些样本点组成的集合称为随机事件，简称事件，常用大写字母A,B,C,⋯A,B,C,\cdotsA,B,C,⋯表示。

比如，在掷一颗骰子中，A=A =A="出现奇数点"是一个事件，即A={1,3,5}A = \{1,3,5\}A={1,3,5}，它是相应样本空间Ω={1,2,3,4,5,6}\Omega = \{1,2,3,4,5,6\}Ω={1,2,3,4,5,6}的一个子集。

事件AAA发生当且仅当AAA中的某个样本点出现了。

由样本空间Ω\OmegaΩ中的单个元素组成的子集称为基本事件，而样本空间Ω\OmegaΩ的最大子集Ω\OmegaΩ称为必然事件。样本空间的最小子集∅\varnothing∅称为不可能事件。

随机变量

用来表示随机现象结果的变量称为随机变量，常用大写字母X,Y,ZX,Y,ZX,Y,Z表示。很多事件都可以采用随机变量表示，表示时应写明随机变量的含义。

注: 随机变量实质上是一个定义在Ω\OmegaΩ上取值于实数集的函数: X:ω→RX: \omega \to RX:ω→R。

例如，设Z=Z =Z="掷一颗骰子出现的点数"，则1,2,3,4,5,6就是随机变量的所有可能的取值，这时：

  • 事件”出现3点“可以用Z=3Z = 3Z=3表示。
  • Z≤6Z \le 6Z≤6是必然事件Ω\OmegaΩ
  • Z=7Z = 7Z=7是不可能事件。

事件域

所谓的事件域，从直观上将就是一个样本空间中某些子集及其运算（并、交、差、对立）结果组合而成的集合类，以后记事件域为F\mathscr{F}F.

为什么要引入事件域呢？这是因为样本空间的某些子集可能不可测(比如勒贝格不可测集)，所以为了避免这种现象，我们没有必要将所有子集都看成事件，只需将我们感兴趣的子集看成事件。

定义1: \quad 设Ω\OmegaΩ是一个样本空间，F\mathscr{F}F为Ω\OmegaΩ的某些子集所组成的集合类. 如果F\mathscr{F}F满足:

  1. Ω∈F\Omega \in \mathscr{F}Ω∈F
  2. 如果A∈FA \in \mathscr{F}A∈F，则A‾∈F\overline{A} \in \mathscr{F}A∈F.
  3. 若Ai=F,i=1,2,⋯A_{i} = \mathscr{F},i = 1,2,\cdotsAi​=F,i=1,2,⋯，则可列并⋃i=1∞Ai∈F\bigcup_{i = 1}^{\infty}A_{i} \in \mathscr{F}⋃i=1∞​Ai​∈F.

则称F\mathscr{F}F为一个事件域，又称为σ\sigmaσ域或者σ\sigmaσ代数。

注:

  1. 事件域的元素应该包括样本空间和空集；
  2. 其次应该保证事件经过并、交、差、对立各种运算后仍然是事件，即其对集合的运算有封闭性。（交的运算可以通过并与对立(德摩根定律)来实现；差的运算可通过对立与交(A−B=AB‾A - B = A \overline{B}A−B=AB)来实现）

例如，若样本空间只包含两个样本点Ω={w1,w2}\Omega = \{w_{1},w_{2}\}Ω={w1​,w2​}，记A={w1}A = \{w_{1}\}A={w1​}，则A‾={w2}\overline{A} = \{w_{2}\}A={w2​}，则其事件域为

{Ω,A,A‾,∅}\{\Omega,A,\overline{A},\varnothing\} {Ω,A,A,∅}

概率的定义

定义2(概率的公理化定义)： \quad 设Ω\OmegaΩ为一个样本空间，F\mathscr{F}F为Ω\OmegaΩ的某些子集组成的一个事件域，如果对任一事件A∈FA \in \mathscr{F}A∈F，定义在F\mathscr{F}F上的一个实值函数P(A)P(A)P(A)满足:

  1. (非负性公理) P(A)≥0P(A) \ge 0P(A)≥0

  2. (正则性公理) P(Ω)=1P(\Omega) = 1P(Ω)=1

  3. (可列可加性公理) 若Ai(i=1,2,⋯ )A_{i}(i = 1,2,\cdots)Ai​(i=1,2,⋯)互不相容，则

    P(⋃i=1∞Ai)=∑i=1∞P(Ai)P(\bigcup_{i = 1}^{\infty}A_{i}) = \sum_{i = 1}^{\infty} P(A_{i}) P(i=1⋃∞​Ai​)=i=1∑∞​P(Ai​)

则称P(A)P(A)P(A)为事件AAA的概率，称三元素(Ω,F,P)(\Omega,\mathscr{F},P)(Ω,F,P)为概率空间。

概率的公理化刻画了概率的本质： 概率是集合（事件）的函数，若在事件域F\mathscr{F}F上给出一个函数，当这个函数能够满足上述三条公理，就被称为概率。

注: 历史上在公理化定义之前出现概率的频率定义、古典定义、几何定义和主观定义都在一定的场合下适用。 例如“贝特朗奇论”在不同的计算方法下会得出不同的计算结果。

概率的性质

性质1: \quad P(∅)=0P(\varnothing) = 0P(∅)=0.

证明:

P(∅)=P(⋃i=1∞∅)=∑i=1∞P(∅)P(\varnothing) = P( \bigcup_{i = 1}^{\infty} \varnothing) = \sum_{i = 1}^{\infty} P(\varnothing) P(∅)=P(i=1⋃∞​∅)=i=1∑∞​P(∅)

又P(∅)≥0P(\varnothing) \ge 0P(∅)≥0

因此，P(∅)=0P(\varnothing) = 0P(∅)=0.

性质2(有限可加性): \quad若有限个事件A1,A2,⋯ ,AnA_{1},A_{2},\cdots,A_{n}A1​,A2​,⋯,An​互不相容，则

P(⋃i=1nAi)=∑i=1nP(Ai)P(\bigcup_{i = 1}^{n} A_{i}) = \sum_{i = 1}^{n} P(A_{i}) P(i=1⋃n​Ai​)=i=1∑n​P(Ai​)

证明: 不妨设An+1=An+2=⋯=∅A_{n+1} = A_{n+2} = \cdots =\varnothingAn+1​=An+2​=⋯=∅，则

P(⋃i=1nAi)=P(⋃i=1∞Ai)=∑i=1∞P(Ai)=∑i=1nP(Ai)+∑i=n+1∞P(∅)=∑i=1nP(Ai)P(\bigcup_{i = 1}^{n} A_{i}) = P(\bigcup_{i = 1}^{\infty} A_{i}) = \sum_{i = 1}^{\infty} P(A_{i}) = \sum_{i = 1}^{n}P(A_{i}) + \sum_{i = n+1}^{\infty}P(\varnothing) = \sum_{i = 1}^{n}P(A_{i}) P(i=1⋃n​Ai​)=P(i=1⋃∞​Ai​)=i=1∑∞​P(Ai​)=i=1∑n​P(Ai​)+i=n+1∑∞​P(∅)=i=1∑n​P(Ai​)

性质3(对立事件的概率公式): \quad 对任一事件AAA，有

P(A‾)=1−P(A)P(\overline{A}) = 1 - P(A) P(A)=1−P(A)

证明:

P(Ω)=P(A∪A‾)=P(A)+P(A‾)=1P(\Omega) = P(A \cup \overline{A}) = P(A) + P(\overline{A}) = 1 P(Ω)=P(A∪A)=P(A)+P(A)=1

所以

P(A‾)=1−P(A)P(\overline{A}) = 1 - P(A) P(A)=1−P(A)

性质4: \quad 若A⊂BA \subset BA⊂B，则

P(B−A)=P(B)−P(A)P(B - A) = P(B) - P(A) P(B−A)=P(B)−P(A)

证明:

由于A⊂BA \subset BA⊂B，所以

A∪(B−A)=BA \cup (B - A) = B A∪(B−A)=B

且

A∩(B−A)=∅A \cap (B - A) = \varnothing A∩(B−A)=∅

因此，根据概率的有限可加性，有

P(B)=P(A∪(B−A))=P(A)+P(B−A)P(B) =P(A \cup (B - A)) = P(A) + P(B - A) P(B)=P(A∪(B−A))=P(A)+P(B−A)

于是

P(B−A)=P(B)−P(A)P(B - A) = P(B) - P(A) P(B−A)=P(B)−P(A)

性质5(任意两个事件差的概率): \quad 对任意的两个事件A,BA,BA,B，有

P(B−A)=P(B)−P(AB)P(B - A) = P(B) - P(AB) P(B−A)=P(B)−P(AB)

证明:

P(B−A)=P(B−AB)=P(B)−P(AB)P(B - A) = P(B - AB) = P(B) - P(AB) P(B−A)=P(B−AB)=P(B)−P(AB)

性质6(加法公式): \quad 对任意两个事件A,BA,BA,B有，

P(A∪B)=P(A)+P(B)−P(AB)P(A \cup B) = P(A) + P(B) - P(AB) P(A∪B)=P(A)+P(B)−P(AB)

对任意nnn个事件A1,A2,⋯ ,AnA_{1},A_{2},\cdots,A_{n}A1​,A2​,⋯,An​，有

P(⋃i=1nAi)=∑i=1nP(Ai)−∑1≤i<j≤nP(AiAj)+⋯+(−1)n−1P(A1A2⋯An)P(\bigcup_{i = 1}^{n}A_{i}) = \sum_{i = 1}^{n}P(A_{i}) - \sum_{1 \le i< j \le n}P(A_{i}A_{j}) + \cdots + (-1)^{n-1}P(A_{1}A_{2} \cdots A_{n}) P(i=1⋃n​Ai​)=i=1∑n​P(Ai​)−1≤i<j≤n∑​P(Ai​Aj​)+⋯+(−1)n−1P(A1​A2​⋯An​)

证明:

P(A∪B)=P(A∪(B−A))=P(A)+P(B−A)=P(A)+P(B)−P(AB)P(A \cup B) = P(A \cup (B - A)) = P(A) + P(B - A) = P(A) + P(B) - P(AB) P(A∪B)=P(A∪(B−A))=P(A)+P(B−A)=P(A)+P(B)−P(AB)

下面用归纳法证明nnn个事件的加法公式，当n=2n = 2n=2时，已经证明成立了。

设nnn个事件的加法公式对n−1n-1n−1成立，则

P(A1∪A2∪⋯∪An)=P(⋃i=1n−1Ai)+P(An)−P((⋃i=1n−1Ai)∩An)\begin{aligned} P(A_{1} \cup A_{2} \cup \cdots \cup A_{n}) & = P(\bigcup_{i = 1}^{n-1}A_{i}) + P(A_{n}) - P((\bigcup_{i = 1}^{n-1}A_{i}) \cap A_{n}) \\ \end{aligned} P(A1​∪A2​∪⋯∪An​)​=P(i=1⋃n−1​Ai​)+P(An​)−P((i=1⋃n−1​Ai​)∩An​)​

然后由归纳假设，将P(⋃i=1n−1Ai)P(\bigcup_{i = 1}^{n-1}A_{i})P(⋃i=1n−1​Ai​)和P((⋃i=1n−1Ai)∩AnP((\bigcup_{i = 1}^{n-1}A_{i}) \cap A_{n}P((⋃i=1n−1​Ai​)∩An​进行展开，经过整理合并即可证明。

条件概率

条件概率的定义

定义3: \quad 设A,BA,BA,B分别是样本空间Ω\OmegaΩ中的两事件，若P(B)>0P(B) > 0P(B)>0，则称

P(A∣B)=P(AB)P(B)P(A|B) = \frac{P(AB)}{P(B)} P(A∣B)=P(B)P(AB)​

为“在BBB发生的条件下AAA的条件概率",简称条件概率。

注: 在计算条件概率P(A∣B)P(A|B)P(A∣B)时，相当于样本空间缩小为ΩB\Omega_{B}ΩB​，计算此时事件AAA的概率。

需要注意的是，条件概率P(A∣B)P(A|B)P(A∣B)是在给定BBB下讨论事件AAA的概率，那么概率的性质对条件概率是否成立呢？我们只要验证条件概率符合概率的公理化定义即可。

性质7: \quad条件概率P(⋅∣B)P(\cdot | B)P(⋅∣B)是概率，即若设P(B)>0P(B) > 0P(B)>0，则

  1. P(A∣B)≥0P(A|B) \ge 0P(A∣B)≥0

  2. P(Ω∣B)=1P(\Omega|B) = 1P(Ω∣B)=1

  3. 若F\mathscr{F}F中的Ai,i=1,2,⋯ ,nA_{i}, i = 1,2,\cdots,nAi​,i=1,2,⋯,n互不相容，则

    P(⋃i=1∞Ai∣B)=∑i=1∞P(Ai∣B)P(\bigcup_{i = 1}^{\infty} A_{i} | B) = \sum_{i = 1}^{\infty} P(A_{i} | B) P(i=1⋃∞​Ai​∣B)=i=1∑∞​P(Ai​∣B)

证明:

1,2点都很容易根据条件概率的定义证明。下面主要证明第3点。

P(⋃i=1∞Ai∣B)=P(A1B∪A2B∪⋯ )P(B)=∑i=1∞P(AiB)P(B)=∑i=1∞P(Ai∣B)P(\bigcup_{i = 1}^{\infty} A_{i} | B) = \frac{P(A_{1}B \cup A_{2}B \cup \cdots )}{P(B)} = \sum_{i = 1}^{\infty} \frac{P(A_{i}B)}{P(B)} = \sum_{i = 1}^{\infty} P(A_{i} | B) P(i=1⋃∞​Ai​∣B)=P(B)P(A1​B∪A2​B∪⋯)​=i=1∑∞​P(B)P(Ai​B)​=i=1∑∞​P(Ai​∣B)

若P(A1A2⋯An)>0P(A_{1} A_{2}\cdots A_{n}) > 0P(A1​A2​⋯An​)>0，则

P(A1A2⋯An)=P(A1)P(A2∣A1)⋯P(An∣A1A2⋯An)P(A_{1} A_{2}\cdots A_{n}) = P(A_{1}) P(A_{2} | A_{1}) \cdots P(A_{n}|A_{1}A_{2} \cdots A_{n}) P(A1​A2​⋯An​)=P(A1​)P(A2​∣A1​)⋯P(An​∣A1​A2​⋯An​)

全概率公式

全概率公式提供了计算复杂事件概率的一条有效途径，即全概率公式将对一复杂事件AAA的概率求解问题转化为了在不同情况或不同原因 BiB_iBi​下发生的简单事件的概率的求和问题。

设B1,B2,⋯ ,BnB_{1},B_{2},\cdots,B_{n}B1​,B2​,⋯,Bn​为样本空间Ω\OmegaΩ的一个分割，即B1,B2,⋯ ,BnB_{1},B_{2},\cdots,B_{n}B1​,B2​,⋯,Bn​互不相容，且⋃i=1nBi=Ω\bigcup_{i = 1}^{n}B_{i} = \Omega⋃i=1n​Bi​=Ω。

则对任一事件AAA有

P(A)=∑i=1nP(Bi)P(A∣Bi)P(A) = \sum_{i = 1}^{n} P(B_{i}) P(A|B_{i}) P(A)=i=1∑n​P(Bi​)P(A∣Bi​)

证明:

P(A)=P(A∩Ω)=P(A∩⋃i=1nBi)=P(⋃i=1n(A∩Bi))=∑i=1nP(ABi)=∑i=1nP(Bi)P(A∣Bi)\begin{aligned} P(A) & = P(A \cap \Omega) \\ & = P(A \cap \bigcup_{i = 1}^{n}B_{i}) \\ & = P(\bigcup_{i = 1}^{n} (A \cap B_{i})) \\ & = \sum_{i = 1}^{n} P(AB_{i}) \\ & = \sum_{i = 1}^{n} P(B_{i}) P(A|B_{i}) \end{aligned} P(A)​=P(A∩Ω)=P(A∩i=1⋃n​Bi​)=P(i=1⋃n​(A∩Bi​))=i=1∑n​P(ABi​)=i=1∑n​P(Bi​)P(A∣Bi​)​

注: Bi,i=1,2,⋯ ,nB_{i},i = 1,2,\cdots ,nBi​,i=1,2,⋯,n的条件可以适当减弱为： Bi,i=1,2,⋯ ,nB_{i}, i = 1,2,\cdots ,nBi​,i=1,2,⋯,n互不相容，且A⊆⋃i=1nBiA \subseteq \bigcup_{i = 1}^{n} B_{i}A⊆⋃i=1n​Bi​

贝叶斯公式

设B1,B2,⋯ ,BnB_{1},B_{2},\cdots,B_{n}B1​,B2​,⋯,Bn​是样本空间Ω\OmegaΩ的一个分割，如果P(A)>0,P(Bi)>0,i=1,2,⋯ ,nP(A) > 0,P(B_{i}) > 0,i = 1,2,\cdots , nP(A)>0,P(Bi​)>0,i=1,2,⋯,n，则

P(Bi∣A)=P(Bi)P(A∣Bi)∑j=1nP(Bj)P(A∣Bj)P(B_{i} | A) = \frac{P(B_{i}) P(A|B_{i})}{\sum_{j = 1}^{n} P(B_{j}) P(A|B_{j})} P(Bi​∣A)=∑j=1n​P(Bj​)P(A∣Bj​)P(Bi​)P(A∣Bi​)​

证明:

P(Bi∣A)=P(ABi)P(A)P(B_{i} | A) = \frac{P(AB_{i})}{P(A)} P(Bi​∣A)=P(A)P(ABi​)​

对分子用乘法公式、分母用全概率公式即可得到贝叶斯公式。

注： 在贝叶斯公式中，有几个约定俗成的名词：

  • P(A∣B)P(A|B)P(A∣B)是已知BBB发生后，AAA的条件概率。也由于得自BBB的取值而被称为AAA的后验概率。
  • P(A)P(A)P(A)是AAA的先验概率（或边缘概率），因为它不考虑任何BBB方面的因素。
  • 对贝叶斯公式的理解: 把事件BiB_{i}Bi​看做是原因，将AAA看做是最后的结果，则贝叶斯公式是在已知结果的条件下，求出原因的概率P(Bi∣A)P(B_{i}|A)P(Bi​∣A)。通过AAA的发生这个新的信息，来对BiB_{i}Bi​的概率作出修正。

独立性的定义

定义4: \quad 如果

P(AB)=P(A)P(B)P(AB) = P(A) P(B) P(AB)=P(A)P(B)

则称事件AAA与事件BBB相互独立，简称AAA与BBB独立，否则称AAA与BBB不独立或相依。

从概率的角度来看，事件AAA的条件概率P(A∣B)P(A|B)P(A∣B)与无条件概率P(A)P(A)P(A)的差别在于: 事件BBB的发生改变了事件AAA发生的概率。 如果事件AAA与事件BBB的发生时相互不影响的，则有P(A∣B)=P(A)P(A|B) = P(A)P(A∣B)=P(A),P(B∣A)=P(B)P(B|A) = P(B)P(B∣A)=P(B)，它们都等价于

P(AB)=P(A)P(B∣A)=P(A)P(B)P(AB) = P(A) P(B|A) = P(A) P(B) P(AB)=P(A)P(B∣A)=P(A)P(B)

在实际问题中，两个事件的独立性往往是根据经验（相互有无影响）来判断的，但在有些问题中，也用上述定义来判断独立性。

试验的独立性

定义6: \quad 设有两个试验E1E_{1}E1​和E2E_{2}E2​，假如试验E1E_{1}E1​的任一结果与试验E2E_{2}E2​的任一结果都是相互独立的事件，则称这两个试验是相互独立的。

例如，掷一枚硬币与掷一颗骰子是相互独立地试验。
