概率论易错题

Posted on 2020-05-04 Edited on 2020-07-09 In Math , 概率论

隐含信息

  1. 已知P(A)=P(B)=P(C)=1/4P(A) = P(B) = P(C) = 1/4P(A)=P(B)=P(C)=1/4，P(AB)=0,P(AC)=P(BC)=1/16P(AB) = 0,P(AC) = P(BC) = 1/16P(AB)=0,P(AC)=P(BC)=1/16. 则A,B,CA,B,CA,B,C中至少发生一个的概率是多少？

    解：

    由概率加法公式，有

    P(A∪B∪C)=P(A)+P(B)+P(C)−(P(AB)+P(AC)+P(BC))+P(ABC)P(A \cup B \cup C) = P(A) + P(B) + P(C) -(P(AB) + P(AC) + P(BC)) + P(ABC) P(A∪B∪C)=P(A)+P(B)+P(C)−(P(AB)+P(AC)+P(BC))+P(ABC)

    题目没有告诉我们P(ABC)P(ABC)P(ABC)的值，这是一个隐含信息。

    由于ABC⊂ABABC \subset ABABC⊂AB，所以P(ABC)≤P(AB)=0P(ABC) \le P(AB) = 0P(ABC)≤P(AB)=0.

    又因为概率的非负性: P(ABC)≥0P(ABC) \ge 0P(ABC)≥0,所以得P(ABC)=0P(ABC) = 0P(ABC)=0.

    将各个数值带入加法公式，得到P(A∪B∪C)=5/8P(A \cup B \cup C) = 5/8P(A∪B∪C)=5/8

  2. 设事件A,B,CA,B,CA,B,C两两独立，满足ABC=∅ABC = \varnothingABC=∅，P(A)=P(B)=P(C)P(A) = P(B) = P(C)P(A)=P(B)=P(C)，且P(A+B+C)=916P(A + B + C) = \dfrac{9}{16}P(A+B+C)=169​，则P(A)=?P(A) = ?P(A)=?

    注意： 这道题可以解出两个可能解，分别是: 14\dfrac{1}{4}41​或者34\dfrac{3}{4}43​，但是要注意隐含信息: P(A)≤P(A+B+C)=916P(A) \le P(A + B + C) = \dfrac{9}{16}P(A)≤P(A+B+C)=169​，因此P(A)=14P(A) = \dfrac{1}{4}P(A)=41​.

概率定义

  1. P(A)=0⇒A=∅P(A) = 0 \Rightarrow A = \varnothingP(A)=0⇒A=∅. 错。同理，P(A)=1⇒A=ΩP(A) = 1 \Rightarrow A = \OmegaP(A)=1⇒A=Ω也是错的。例如，在0到1这个线段内，随机的选择一个点，这个点为有理数的概率为0，这个点为无理数的概率为1. 因为无理数的测度为1，有理数的测度为0. 但是我随机选的这个点有可能不是无理数，它是有可能是有理数的。在这个例子里，概率为0的事件并不是不可能事件，概率为1的事件并不是必然事件。

  2. P(AB)=0⇒A,BP(AB) = 0 \Rightarrow A,BP(AB)=0⇒A,B互不相容。 主要错在误以为AB=∅AB = \varnothingAB=∅，和第1点类似。同时，也需要弄清楚不相容的概念： 即A∩B=∅A \cap B = \varnothingA∩B=∅.

  3. 概率密度函数的定义和概率混淆。概率密度函数pX(x)p_{X}(x)pX​(x)不是概率，但是乘以微元dxdxdx就可以得到小区间(x,xdx)(x,x_dx)(x,xd​x)上概率的近似值，即

    pX(x)dx≈P(x<X<x+dx)p_{X}(x) dx \approx P(x < X< x + dx) pX​(x)dx≈P(x<X<x+dx)
