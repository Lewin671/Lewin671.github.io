可逆矩阵

定义4: \quad 令AAA是数域FFF上的一个nnn阶矩阵. 若是存在FFF上nnn阶矩阵BBB，使得

AB=BA=IAB = BA = I AB=BA=I

关于可逆矩阵，由如下的性质:

  1. 若nnn阶方阵AAA可逆，则其转置矩阵ATA^{T}AT也可逆，且

    (AT)−1=(A−1)T(A^{T})^{-1} = (A^{-1})^{T} (AT)−1=(A−1)T

    AT(A−1)T=(A−1A)T=IT=IA^{T}(A^{-1})^{T} = (A^{-1}A)^{T} = I^{T} = I AT(A−1)T=(A−1A)T=IT=I

    并且

    (A−1)TAT=(AA−1)T=IT=I(A^{-1})^{T} A^{T} = (A A^{-1})^{T} = I^{T} = I (A−1)TAT=(AA−1)T=IT=I

    由可逆矩阵定义可知命题得证。

  2. 若矩阵AAA可逆，并且满足AB=OAB = OAB=O（或者BA=OBA = OBA=O），则B=OB = OB=O.

    证明:

    B=IB=A−1AB=A−1O=OB = IB = A^{-1}AB = A^{-1} O = O B=IB=A−1AB=A−1O=O

    推论: 若矩阵AAA可逆，并且AB=ACAB = ACAB=AC（或者BA=CABA = CABA=CA），则B=CB = CB=C.

    证明:

    A(B−C)=AB−AC=OA(B-C) = AB-AC = O A(B−C)=AB−AC=O

    并且AAA可逆，所以B−C=OB - C = OB−C=O，即B=CB = CB=C.

  3. 若AAA与BBB均为nnn阶可逆矩阵，则ABABAB也可逆，并且:

    (AB)−1=B−1A−1(AB)^{-1} = B^{-1}A^{-1} (AB)−1=B−1A−1

    证明:

    (AB)(B−1A−1)=(B−1A−1)(AB)=I(AB)(B^{-1}A^{-1}) = (B^{-1}A^{-1})(AB) = I (AB)(B−1A−1)=(B−1A−1)(AB)=I

    由可逆矩阵定义可知命题得证。

    推广:

    (A1A2⋯Ak)−1=Ak−1Ak−1−1⋯A1−1(A_{1}A_{2} \cdots A_{k})^{-1} = A_{k}^{-1}A_{k-1}^{-1} \cdots A_{1}^{-1} (A1​A2​⋯Ak​)−1=Ak−1​Ak−1−1​⋯A1−1​
