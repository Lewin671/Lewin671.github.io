常微分方程的数值解法

这里主要讨论一阶常微分方程初值问题

{y′=f(x,y)y(x0)=y0(1)\begin{cases} y' = f(x,y) \\ y(x_{0}) = y_{0} \end{cases} \tag{1} {y′=f(x,y)y(x0​)=y0​​(1)

的数值解法.

记步长hhh，记xn=x0+nhx_{n} = x_{0} + nhxn​=x0​+nh，按照一定的递推公式依次求得各节点xnx_{n}xn​上解函数值y(xn)y(x_{n})y(xn​)的近似值yny_{n}yn​.

欧拉公式及其改进

欧拉公式和隐式欧拉公式

由向前差商公式

y′(xn)≈f(xn+1)−f(xn)h(2)y'(x_{n}) \approx \frac{f(x_{n+1}) - f(x_{n})}{h} \tag{2} y′(xn​)≈hf(xn+1​)−f(xn​)​(2)

得到

f(xn+1)≈f(xn)+y′(xn)h(3)f(x_{n+1}) \approx f(x_{n}) + y'(x_{n})h \tag{3} f(xn+1​)≈f(xn​)+y′(xn​)h(3)

用yny_{n}yn​来近似y(xn)y(x_{n})y(xn​)，那么

yn+1=yn+hf(xn,yn),n=0,1,⋯(4)y_{n+1} = y_{n} + hf(x_{n},y_{n}), n = 0,1,\cdots \tag{4} yn+1​=yn​+hf(xn​,yn​),n=0,1,⋯(4)

同理，根据向后差商公式可以推导得到

yn+1=yn+hf(xn+1,yn+1),n=0,1,⋯(5)y_{n+1} = y_{n} + hf(x_{n+1},y_{n+1}), n = 0,1,\cdots \tag{5} yn+1​=yn​+hf(xn+1​,yn+1​),n=0,1,⋯(5)

其中(4)(4)(4)叫做欧拉公式，而(5)(5)(5)叫做隐式欧拉公式.

定义1 \quad 对于求解式(1)(1)(1)的某差分格式，hhh为步长. 假设y1,y2,⋯ ,yny_{1},y_{2},\cdots,y_{n}y1​,y2​,⋯,yn​是准确的，称

εn+1=y(xn+1)−yn+1(6)\varepsilon_{n+1} = y(x_{n+1}) - y_{n+1} \tag{6} εn+1​=y(xn+1​)−yn+1​(6)

为该差分格式的局部截断. 当εn+1=O(hp+1)\varepsilon_{n+1} = O(h^{p+1})εn+1​=O(hp+1)时，称该差分格式具有ppp阶精度.

根据定义1，可以推导出欧拉公式和隐式欧拉公式都是一阶精度.

改进欧拉公式

对(1)(1)(1)式两边在区间[xn,xn+1][x_{n},x_{n+1}][xn​,xn+1​]进行积分，得到

y(xn+1)=y(xn)+∫xnxn+1f(x,y(x))dx(7)y(x_{n+1}) = y(x_{n}) + \int_{x_{n}}^{x_{n+1}}f(x,y(x)) dx \tag{7} y(xn+1​)=y(xn​)+∫xn​xn+1​​f(x,y(x))dx(7)

应用数值积分公式求解(7)(7)(7)式中的积分，可以得到相应的差分格式. 例如，由左矩形公式

∫abf(x)≈(b−a)f(a)(8)\int_{a}^{b} f(x) \approx (b-a)f(a) \tag{8} ∫ab​f(x)≈(b−a)f(a)(8)

可以导出

y(xn+1)≈y(xn)+(xn+1−xn)f(xn)(9)y(x_{n+1}) \approx y(x_{n}) + (x_{n+1} - x_{n}) f(x_{n}) \tag{9} y(xn+1​)≈y(xn​)+(xn+1​−xn​)f(xn​)(9)

即

yn+1=yn+hf(xn,yn)(10)y_{n+1} = y_{n} + hf(x_{n},y_{n}) \tag{10} yn+1​=yn​+hf(xn​,yn​)(10)

此外，由梯形公式

∫abf(x)dx≈b−a2(f(a)+f(b))(11)\int_{a}^{b}f(x)dx \approx \frac{b-a}{2}(f(a) + f(b)) \tag{11} ∫ab​f(x)dx≈2b−a​(f(a)+f(b))(11)

可以导出下面的差分格式

yn+1=yn+h2[f(xn,yn)+f(xn+1,yn+1)](12)y_{n+1} = y_{n} + \frac{h}{2}[f(x_{n},y_{n})+f(x_{n+1},y_{n+1})] \tag{12} yn+1​=yn​+2h​[f(xn​,yn​)+f(xn+1​,yn+1​)](12)

式(12)(12)(12)也称为梯形公式. 可以证明,(12)(12)(12)的局部截断误差为

yxn+1−yn+1=O(h3)(13)y_{x_{n+1}} -y_{n+1} = O(h^{3}) \tag{13} yxn+1​​−yn+1​=O(h3)(13)

这说明梯形公式具有二阶精度.

但是梯形公式是一个隐格式，不方便使用。因此我们先用(4)(4)(4)对yn+1y_{n+1}yn+1​进行计算，并将结果记做pn+1p_{n+1}pn+1​，再带入(12)(12)(12)，可以得到"预报，校正"形式的差分格式:

{pn+1=yn+f(xn,yn)yn+1=yn+h2[f(xn,yn)+f(xn+1,pn+1)](14)\begin{cases} p_{n+1} = y_{n} + f(x_{n},y_{n}) \\ y_{n+1} = y_{n} + \frac{h}{2}[f(x_{n},y_{n}) + f(x_{n+1},p_{n+1})] \end{cases} \tag{14} {pn+1​=yn​+f(xn​,yn​)yn+1​=yn​+2h​[f(xn​,yn​)+f(xn+1​,pn+1​)]​(14)

式(14)(14)(14)称为改进欧拉公式，可以证明其精度也是二阶的.
