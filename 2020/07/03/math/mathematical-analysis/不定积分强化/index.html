不定积分强化

Posted on 2020-07-03 Edited on 2020-07-04 In Math , 数学分析

之前讲了一篇关于不定积分的文章，主要是介绍不定积分的定义，以及不定积分的相关理论。这篇文章主要侧重于计算不定积分及其技巧。

基本不定积分的计算

多项式分式

对于多项式分式，需要使用部分分式积分法，将有理函数分解成许多次数较低有理函数和的形式，来降低分子或分母多项式的次数。分解后的分式满足以下条件：

  • 分式的分母需为不可约多项式（irreducible polynomial）或其乘幂。
  • 分式的分子多项式次数需比其分母多项式次数要低

三角函数

对于三角函数的乘积，一般通过积化和差公式来化简积分。

例如：

∫cos⁡3xsin⁡xdx=12∫(sin⁡4x−sin⁡2x)dx=12(−14cos⁡4x+12cos⁡2x)+C\int \cos 3x \sin x dx = \frac{1}{2} \int (\sin 4x - \sin 2x) dx = \frac{1}{2}(-\frac{1}{4} \cos 4x + \frac{1}{2} \cos 2x ) + C∫cos3xsinxdx=21​∫(sin4x−sin2x)dx=21​(−41​cos4x+21​cos2x)+C

∫dxcos⁡2xsin⁡2x=∫sin⁡2x+cos⁡2xcos⁡2xsin⁡2xdx=∫(sec⁡2x+csc⁡2x)dx=tan⁡x−cot⁡x+C\begin{aligned} \int \frac{dx}{\cos^2 x \sin^2 x} &= \int \frac{\sin^2 x + \cos^2 x}{\cos^2x \sin^2x} dx \\ & = \int(\sec^2 x + \csc^2 x)dx \\ & = \tan x - \cot x + C \end{aligned}∫cos2xsin2xdx​​=∫cos2xsin2xsin2x+cos2x​dx=∫(sec2x+csc2x)dx=tanx−cotx+C​

分段函数

在计算分段函数的不定积分时，应该分别求出函数的各个分段在相应区间内的不定积分，然后再勘察被积函数在分段点处的连续性。

  • 如果连续，则在包含该点的区间内它的原函数存在，这时只需要根据原函数的连续性（可导必然连续），确定各个区间之间常数的关系。
  • 如果该分段点是被积函数的第一类间断点，那么在包含该点的区间内不存在原函数，这时它们的积分常数是互相独立的。

积分换元法

第一类积分换元法（配凑法）

配凑法一般通过观察，能否配凑？配凑后能否化简？技巧性比较少。不过，有些式子需要处理后再进行配凑，比如∫cos⁡2xdx\int \cos^2 xdx∫cos2xdx需要降幂后再配凑。

第二积分换元法

在第二积分换元法中，换元一般有如下规律：

  1. 如被积函数中含有x2−a2\sqrt{x^2 - a^2}x2−a2​可试x=asec⁡tx = a \sec tx=asect
  2. 如被积函数中含有a2−x2\sqrt{a^2 - x^2}a2−x2​可试x=asin⁡tx = a \sin tx=asint或者x=acos⁡tx = a \cos tx=acost.
  3. 如被积函数中含有a2+x2\sqrt{a^2 + x^2}a2+x2​可试x=atan⁡tx = a \tan tx=atant.
  4. 如被积函数中含有ax+bn\sqrt[n]{ax + b}nax+b​可试t=ax+bnt = \sqrt[n]{ax + b}t=nax+b​
  5. 如被积函数中含有ax+bcx+dn\sqrt[n]{\frac{ax + b}{cx + d}}ncx+dax+b​​可试令t=ax+bcx+dnt = \sqrt[n]{\frac{ax + b}{cx + d}}t=ncx+dax+b​​
  6. 如被积函数中同时含有ax+bcx+dm\sqrt[m]{\frac{ax + b}{cx + d}}mcx+dax+b​​和ax+bcx+dn\sqrt[n]{\frac{ax + b}{cx + d}}ncx+dax+b​​，可试令ax+bcx+dk=t\sqrt[k]{\frac{ax + b}{cx + d}} = tkcx+dax+b​​=t，其中kkk为m,nm,nm,n的最小公倍数。
  7. 如被积函数为商形式，若分子次数比分母小，可试用t=1xt = \frac{1}{x}t=x1​.

分部积分

分部积分公式:

∫udv=uv−∫vdu\int u dv = uv - \int v du∫udv=uv−∫vdu

选取uuu的原则，一般按照“反对幂指三”的优先级来选取，即vvv比较容易就可以求得。

注:

反： 反三角函数

对： 对数函数

幂： 幂函数

指： 指数函数

三： 三角函数
