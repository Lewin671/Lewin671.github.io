不定积分强化

Posted on 2020-07-03 Edited on 2020-11-02 In Math , 数学分析

之前讲了一篇关于不定积分的文章，主要是介绍不定积分的定义，以及不定积分的相关理论。这篇文章主要侧重于计算不定积分及其技巧。

基本不定积分的计算

常用的不定积分公式

∫sec⁡xdx=ln⁡(sec⁡x+tan⁡x)+C\int \sec x dx = \ln(\sec x + \tan x) + C ∫secxdx=ln(secx+tanx)+C

证明:

∫sec⁡xdx=∫sec⁡x(sec⁡x+tan⁡x)sec⁡x+tan⁡xdx=∫d(sec⁡x+tan⁡x)sec⁡x+tan⁡x=ln⁡(sec⁡x+tan⁡x)+C\begin{aligned} \int \sec x dx & = \int \dfrac{\sec x(\sec x + \tan x)}{\sec x + \tan x} dx\\ &= \int \dfrac{\mathrm{d}(\sec x + \tan x)}{\sec x + \tan x} \\ & = \ln(\sec x + \tan x) + C \end{aligned} ∫secxdx​=∫secx+tanxsecx(secx+tanx)​dx=∫secx+tanxd(secx+tanx)​=ln(secx+tanx)+C​

有理函数

有理函数是指由两个多项式的商所表示的函数，其一般形式为

R(x)=P(x)Q(x)=α0xn+⋯+αnβ0xm+⋯+βmR(x) = \dfrac{P(x)}{Q(x)} = \dfrac{\alpha_{0} x^{n} + \cdots + \alpha_{n}}{\beta_{0} x^{m} + \cdots + \beta_{m}} R(x)=Q(x)P(x)​=β0​xm+⋯+βm​α0​xn+⋯+αn​​

若m>nm > nm>n，则称它为真分式，否则称为假分式。

若R(x)R(x)R(x)是一个假分式，则可以根据多项式除法将R(x)R(x)R(x)化为一个多项式和一个真分式的和。因此，我们只需研究真分式的不定积分，故下面基于R(x)R(x)R(x)为一个真分式来讨论。

根据代数知识，有理真分式必定可以表示成若干个部分分式之和，因此问题归结为求那些部分分式的不定积分。由于部分分式的分解过程在考试中运算量比较大，所以这里不再进行赘述，可以参考华东师范大学的《数学分析》第192页，也可以参考这个链接： 部分分式教程链接。

只有在没有其他办法的情况下，才使用部分分式分解的办法！

而我们在平时中用的比较多的一般是加项减项或凑微分的方法来解决这个问题。

∫x+5x2−6x+13dx=12∫d(x2−6x+13)x2−6x+13+8∫d(x−3)(x−3)2+4=12ln⁡∣x2−6x+13∣+4arctan⁡x−32+C\begin{aligned} \int \dfrac{x + 5}{x^2 - 6x + 13} \mathrm{d}x & = \dfrac{1}{2} \int \dfrac{\mathrm{d}(x^2 - 6x + 13)}{x^2 - 6x + 13} + 8\int \dfrac{\mathrm{d}(x-3)}{(x - 3)^2 + 4} \\ & = \dfrac{1}{2} \ln|x^2 - 6x + 13| + 4 \arctan \dfrac{x - 3}{2} + C \end{aligned} ∫x2−6x+13x+5​dx​=21​∫x2−6x+13d(x2−6x+13)​+8∫(x−3)2+4d(x−3)​=21​ln∣x2−6x+13∣+4arctan2x−3​+C​

分段函数

在计算分段函数的不定积分时，应该分别求出函数的各个分段在相应区间内的不定积分，然后再考察被积函数在分段点处的连续性。

  • 如果连续，则在包含该点的区间内它的原函数存在，这时只需要根据原函数的连续性（可导必然连续），确定各个区间之间常数的关系。
  • 如果该分段点是被积函数的第一类间断点，那么在包含该点的区间内不存在原函数，这时它们的积分常数是互相独立的。

注: 若f(x)f(x)f(x)在区间III上有第一类间断点，则fff不存在原函数.

设f(x)=∣x−1∣f(x) = |x- 1|f(x)=∣x−1∣，则f(x)f(x)f(x)连续，所以可积。

积分换元法

第一类积分换元法（配凑法）

配凑法一般通过观察，能否配凑？配凑后能否化简？技巧性比较少。不过，有些式子需要处理后再进行配凑，比如∫cos⁡2xdx\displaystyle{\int \cos^2 x\mathrm{d}x}∫cos2xdx需要降幂后再配凑。

第二积分换元法

在第二积分换元法中，换元一般有如下规律：

  1. 如被积函数中含有x2−a2\sqrt{x^2 - a^2}x2−a2​可试x=asec⁡tx = a \sec tx=asect
  2. 如被积函数中含有a2−x2\sqrt{a^2 - x^2}a2−x2​可试x=asin⁡tx = a \sin tx=asint或者x=acos⁡tx = a \cos tx=acost.
  3. 如被积函数中含有a2+x2\sqrt{a^2 + x^2}a2+x2​可试x=atan⁡tx = a \tan tx=atant.
  4. 如被积函数中含有ax+bn\sqrt[n]{ax + b}nax+b​可试t=ax+bnt = \sqrt[n]{ax + b}t=nax+b​
  5. 如被积函数中含有ax+bcx+dn\sqrt[n]{\frac{ax + b}{cx + d}}ncx+dax+b​​可试令t=ax+bcx+dnt = \sqrt[n]{\frac{ax + b}{cx + d}}t=ncx+dax+b​​
  6. 如被积函数中同时含有ax+bcx+dm\sqrt[m]{\frac{ax + b}{cx + d}}mcx+dax+b​​和ax+bcx+dn\sqrt[n]{\frac{ax + b}{cx + d}}ncx+dax+b​​，可试令ax+bcx+dk=t\sqrt[k]{\frac{ax + b}{cx + d}} = tkcx+dax+b​​=t，其中kkk为m,nm,nm,n的最小公倍数。
  7. 如被积函数为商形式，若分子次数比分母小，可试用t=1xt = \frac{1}{x}t=x1​.
