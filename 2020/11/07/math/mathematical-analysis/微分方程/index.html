微分方程

Posted on 2020-11-07 Edited on 2020-11-10 In Math , 数学分析

微分方程（英语：Differential equation，DE）是一种数学方程，用来描述某一类函数与其导数之间的关系。微分方程的解是一个符合方程的函数。而在初等数学的代数方程里，其解是常数值。

微分方程的应用十分广泛，可以解决许多与导数有关的问题。物理中许多涉及变力的运动学、动力学问题，如空气的阻力为速度函数的落体运动等问题，很多可以用微分方程求解。此外，微分方程在化学、工程学、经济学和人口统计等领域都有应用。

微分方程的基本概念

一般地，凡表示未知函数、未知函数的导数与自变量之间的关系的方程，叫做微分方程，有时也简称方程。微分方程中所出现的未知函数的最高阶导数的阶数，叫做微分方程的阶。

例如，方程

x3y′′′+x2y′′−4xy′=3x2x^{3} y''' + x^{2}y'' - 4xy' = 3x^{2} x3y′′′+x2y′′−4xy′=3x2

是三阶微分方程。

一般地，nnn阶微分方程的形式是

F(x,y,y′,⋯ ,y(n))=0F(x,y,y',\cdots,y^{(n)}) = 0 F(x,y,y′,⋯,y(n))=0

这里需要指出的是，y(n)y^{(n)}y(n)是必须出现的，而x,y,⋯ ,y(n−1)x,y,\cdots,y^{(n-1)}x,y,⋯,y(n−1)等变量可以不出现。例如nnn阶微分方程

y(n)+1=0y^{(n)} + 1 = 0 y(n)+1=0

微分方程的解: 带入微分方程中成为恒等式的函数。

如果微分方程的解中含有任意常数，且（独立的）任意常数的个数与微分方程的阶数相同，这样的解叫做微分方程的通解。

确定了通解中的任意常数后，就得到微分方程的特解。

微分方程解的图形是一条曲线，叫做微分方程的积分曲线。

一阶微分方程

这里讨论一阶微分方程

dydx=f(x,y)\dfrac{dy}{dx} = f(x,y) dxdy​=f(x,y)

的一些解法。

一阶微分方程有时也写成如下的对称形式:

P(x,y)dx+Q(x,y)dy=0P(x,y) dx + Q(x,y) dy = 0 P(x,y)dx+Q(x,y)dy=0

可分离变量的一阶微分方程

如果一个一阶微分方程可以写成

g(y)dy=f(x)dxg(y) dy = f(x)dx g(y)dy=f(x)dx

的形式，就是说，能把微分方程写成一端只含yyy的函数和dydydy，另一端只含xxx的函数和dxdxdx，那么原方程就称为可分离变量的微分方程。

然后两端积分

∫g(y)dy=∫f(x)dx\int g(y) dy = \int f(x) dx ∫g(y)dy=∫f(x)dx

即可求得原方程通解。

齐次方程

如果一阶方程可以化成

dydx=f(yx)\dfrac{dy}{dx} = f(\dfrac{y}{x}) dxdy​=f(xy​)

的形式，那么就称这个方程为齐次方程。

例如

(xy−y2)dx−(x2−2xy)dy=0(xy - y^{2}) dx - (x^{2} - 2xy)dy = 0 (xy−y2)dx−(x2−2xy)dy=0

是齐次方程，因为它可以化成

dydx=xy−y2x2−2xy\dfrac{dy}{dx} = \dfrac{xy - y^{2}}{x^{2} - 2xy} dxdy​=x2−2xyxy−y2​

即

dydx=yx−(yx)21−2yx\dfrac{dy}{dx} = \dfrac{\dfrac{y}{x} - \left(\dfrac{y}{x}\right)^{2}}{1 - 2\dfrac{y}{x}} dxdy​=1−2xy​xy​−(xy​)2​

求解该方程的方法是做变量代换yx=u\dfrac{y}{x} = uxy​=u，则y=ux,dydx=u+xdudxy = ux, \dfrac{dy}{dx} = u + x \dfrac{du}{dx}y=ux,dxdy​=u+xdxdu​，带入原方程即可得到可分离变量的微分方程

duf(u)−u=dxx\dfrac{du}{f(u) - u} = \dfrac{dx}{x} f(u)−udu​=xdx​

然后求解。

将u(x)u(x)u(x)回代yyy，得到一阶线性非齐次方程通解

y=e−∫P(x)dx(∫Q(x)e∫P(x)dx+C)y = e^{-\int P(x) dx} \left( \int Q(x) e^{\int P(x) dx} + C \right) y=e−∫P(x)dx(∫Q(x)e∫P(x)dx+C)

伯努利方程

方程

dydx+P(x)y=Q(x)yn,(n≠0,1)\dfrac{dy}{dx} + P(x) y = Q(x)y^{n}, (n \not={0,1}) dxdy​+P(x)y=Q(x)yn,(n​=0,1)

叫做伯努利方程。 当n=0,1n = 0,1n=0,1时是线性方程。当n≠0,1n\not={0,1}n​=0,1时，这个方程不是线性方程，但是通过变量代换，便可以把它化为线性方程。

对伯努利方程两边同时除于yny^{n}yn，得到

y−ny′+P(x)y−n+1=Q(x)y^{-n} y' + P(x) y^{-n+1} = Q(x) y−ny′+P(x)y−n+1=Q(x)

令z=y1−nz = y^{1-n}z=y1−n，于是

dzdx=(1−n)y−ndydx\dfrac{dz}{dx} = (1-n) y^{-n} \dfrac{dy}{dx} dxdz​=(1−n)y−ndxdy​

带入上式得到

dzdx+(1−n)P(x)z=(1−n)Q(x)\dfrac{dz}{dx} + (1-n)P(x)z = (1-n)Q(x) dxdz​+(1−n)P(x)z=(1−n)Q(x)

这就化成了线性方程了。
