实数的完备性

Posted on 2020-06-08 In Math , 数学分析

首先，我们先从直观上了什么是实数的完备性（有时候也称实数的连续性）。实数的完备性意味着实数轴上没有“间隙”。这是实数区别于有理数的特点，有理数在实数轴上是有间隙的，即无理数。

实数的完备性公理有一组等价的命题，完备性的定义方式与实数的构造方式相关。在确立之一为公理后，其余皆为完备性公理的等价定理。

区间套定理

定义1: \quad 设闭区间列{[an,bn]}\{[a_{n}, b_{n}]\}{[an​,bn​]}具有如下性质:

  • [an,bn]⊃[an+1,bn+1],n=1,2,⋯[a_{n},b_{n}] \supset [a_{n+1},b_{n+1}], n = 1,2,\cdots[an​,bn​]⊃[an+1​,bn+1​],n=1,2,⋯
  • (bn−an)→0(n→∞)(b_{n} - a_{n}) \to 0(n \to \infty)(bn​−an​)→0(n→∞)

则称{[an,bn}\{[a_{n},b_{n}\}{[an​,bn​}为闭区间套，或者简称区间套。

所以，根据区间套的定义，有

a1≤a2≤⋯≤an≤⋯≤bn≤⋯≤b2≤b1(1)a_{1} \le a_{2} \le \cdots \le a_{n} \le \cdots \le b_{n} \le \cdots \le b_{2} \le b_{1} \tag{1} a1​≤a2​≤⋯≤an​≤⋯≤bn​≤⋯≤b2​≤b1​(1)

定理1(区间套定理):\quad 若{[an,bn]}\{[a_{n}, b_{n}]\}{[an​,bn​]}是一个闭区间套，则在实数系中存在唯一的一点ξ\xiξ，使得ξ∈[an,bn],n=1,2,⋯\xi \in [a_{n},b_{n}], n = 1,2,\cdotsξ∈[an​,bn​],n=1,2,⋯，即

证明: 由(1)(1)(1)式，{an}\{a_{n}\}{an​}为递增有界数列，根据单调有界定理，{an}\{a_{n}\}{an​}有极限ξ\xiξ，且

设ξ′\xi'ξ′也满足
