谓词逻辑

Posted on 2020-02-15 Edited on 2020-04-27 In Math , Discrete Mathematics

Introdunction

命题逻辑还无法表示所有的数学语言和自然语言，比如著名的苏格拉底论证就是如此：“所有的人总是要死的。因为苏格拉底是人，所以苏格拉底是要死的”。这个推理是公认的真命题。在命题逻辑中，符号化表示为：

令ppp: 所有人会死，qqq: 苏格拉底是人, rrr： 苏格拉底会死，所以(p∧q)→r(p \wedge q) \to r(p∧q)→r.

但是我们无法通过命题逻辑来证明它的正确性。所以这里我们将会引入谓词逻辑。

Predicates

在数学陈述中，经常会出现一些包含变量的陈述，当这些变量没有被赋值的时候，我们无法判断数学陈述的真假。但是，一旦我们给所有的变量赋值，数学陈述就变成了命题。

比如: "xxx is greater than 3" 有两个部分。第一部分是xxx，它是这句话的主语；第二部分是谓词（predicate) “is greater than 3"，它指出了这句话的主语所具有的属性。我们把整句话"xxx is greater than 3"用一个命题函数P(x)P(x)P(x)来表示。一旦我们给变量xxx赋值，P(x)P(x)P(x)就变成了一个命题。

例如: Let P(x)P(x)P(x) denote the statement "x>3x > 3x>3. What are the truth values of P(4)P(4)P(4) and P(2)P(2)P(2)?

解:

令x=4x = 4x=4，我们就可以得到P(4)P(4)P(4)，即4>34 > 34>3，很明显这是一个真命题。

令x=2x = 2x=2，我们就可以得到P(2)P(2)P(2)，即2>32 > 32>3，很明显这是一个假命题。

我们也有不止一个变量的陈述。比如： “x=y+3x = y + 3x=y+3”。我们可以用Q(x,y)Q(x,y)Q(x,y)来表示这一个陈述，其中xxx和yyy都是变量，QQQ是谓词。

一般地，包含nnn个变量x1,x2,⋯ ,xnx_{1},x_{2},\cdots,x_{n}x1​,x2​,⋯,xn​的陈述可以用P(x1,x2,⋯ ,xn)P(x_{1},x_{2},\cdots,x_{n})P(x1​,x2​,⋯,xn​)表示，我们把语句P(x1,x2,⋯ ,xn)P(x_{1},x_{2},\cdots,x_{n})P(x1​,x2​,⋯,xn​)叫做PPP在nnn元变量(x1,x2,⋯ ,xn)(x_{1},x_{2},\cdots,x_{n})(x1​,x2​,⋯,xn​)的值，其中PPP叫做n元谓词。

比如，命题”武汉位于北京和广州之间“就可以用谓词表达式表达为: Q(a,b,c)Q(a,b,c)Q(a,b,c)，其中Q(x,y,z)Q(x,y,z)Q(x,y,z)表示xxx位于yyy和zzz之间，aaa表示武汉,bbb表示北京，ccc表示广州。

Quantifies

除了通过对命题函数中的所有变量赋值，通过量化(quantification)也可以从命题函数产生命题。量化表示谓词在一定范围的事物上成立的程度。这里主要处理两种量化： 全称量化和存在量化。处理谓词和量词的领域成为谓词演算(predicate calculus)。

The Universal Quantifier

许多数学的陈述断言在一个特定的论域(domain of discourse)下，一个变量的所有值都满足某种属性，这样的陈述就可以通过全称量化来表达。注意，当论域改变时，全称量化的意义也就改变了，所以当使用一个全称量词的时候，必须要指定一个论域。

定义1 \quad 命题函数P(x)P(x)P(x)的全称量化是命题"P(x)P(x)P(x) for all values of xxx in the domain."，这里用符号∀xP(x)\forall x P(x)∀xP(x)来表示P(x)P(x)P(x)的全称量化，读作"for all xP(x)x P(x)xP(x)"或者"for every xP(x)xP(x)xP(x)". ∀\forall∀就叫做全称量词。能使得P(x)P(x)P(x)为假的元素称为∀xP(x)\forall x P(x)∀xP(x)的反例(counterexample)。

例如，假设P(x)P(x)P(x)表示"x+1>xx + 1 > xx+1>x"，那么全称量化∀xP(x)\forall x P(x)∀xP(x)的真值是什么？其中论域是所有的实数。

解： 因为对于所有的实数P(x)P(x)P(x)都成立，所有全称量化∀xP(x)\forall x P(x)∀xP(x)为真。

注意，一般来说，我们暗含假定量词的所有论域都不为空，如果论域为空，那么我们认为∀xP(x)\forall x P(x)∀xP(x)为真，因为这里没有反例使得P(x)P(x)P(x)为假。

当论域的元素可以被列举时，比如论域的元素为x1,x2,⋯ ,xnx_{1},x_{2},\cdots,x_{n}x1​,x2​,⋯,xn​，那么

∀xP(x)≡P(x1)∧P(x2)∧⋯∧P(xn)\forall x P(x) \equiv P(x_{1}) \wedge P(x_{2}) \wedge \cdots \wedge P(x_{n}) ∀xP(x)≡P(x1​)∧P(x2​)∧⋯∧P(xn​)

因为右边的合取式为真时当且仅当P(x1),P(x2),⋯ ,P(xn)P(x_{1}),P(x_{2}),\cdots,P(x_{n})P(x1​),P(x2​),⋯,P(xn​)都为真。

例如，假定P(x)P(x)P(x)为x2<10x^{2} < 10x2<10，论域为所有不超过4的正整数，那么∀xP(x)\forall x P(x)∀xP(x)的真值是什么？

解： 因为不超过4的正整数可以被列举出来，即1，2，3，4，所以

∀xP(x)≡P(1)∧P(2)∧P(3)∧P(4)\forall x P(x) \equiv P(1) \wedge P(2) \wedge P(3) \wedge P(4) ∀xP(x)≡P(1)∧P(2)∧P(3)∧P(4)

而P(4)P(4)P(4)为42<104^{2} < 1042<10为假，所以∀xP(x)\forall x P(x)∀xP(x)为假。

The Existential Quantifier

许多的数学陈述断言存在一个元素有特定的属性，这样的陈述可以通过存在量化来表达。

定义2 \quad 命题函数P(x)P(x)P(x)的存在量化(existential quantification)是命题”There exists an element xxx in the domain such that P(x)P(x)P(x)“。 我们用∃xP(x)\exist x P(x)∃xP(x)来表示P(x)P(x)P(x)的存在量化，∃\exist∃称为存在量词。

和全称量化类似，存在量化被使用的时候，我们也必须指定论域，因为不用的论域与导致存在量化有不一样的意义。

例如，设P(x)P(x)P(x)表示x>3x > 3x>3，论域为所有的实数，那么∃xP(x)\exist x P(x)∃xP(x)的真值是什么？

解： 因为444是属于论域之内的，而且P(4)P(4)P(4)为真，所以∃xP(x)\exist x P(x)∃xP(x)为真。

注意，如果论域为空，我们认为∃xP(x)\exist x P(x)∃xP(x)为假，因为不存在xxx使得P(x)P(x)P(x)为真。

如果论域的元素可以被列举，假设为x1,x2,⋯ ,xnx_{1},x_{2},\cdots,x_{n}x1​,x2​,⋯,xn​，那么

∃xP(x)≡P(x1)∨P(x2)∨⋯∨P(xn)\exist x P(x) \equiv P(x_{1}) \vee P(x_{2}) \vee \cdots \vee P(x_{n}) ∃xP(x)≡P(x1​)∨P(x2​)∨⋯∨P(xn​)

Quantifiers with Ristricted Domains

缩写的符号常常用于限制量词的论域。一个变量所必须满足的条件在量词的后面被包括起来。比如∀x<0(x2>0)\forall x < 0 (x^{2} > 0)∀x<0(x2>0)表示对于每一个小于0的实数，都有x2>0x^{2} > 0x2>0，即负数的平方是正数。

类似地，∀y≠0(y3≠0)\forall y \not={0} (y^{3} \not={0})∀y​=0(y3​=0)表示对于每一个不为零的数yyy，都有y3≠0y^{3} \not={0}y3​=0，即非零实数的三次方不为零。

注意，全称量化的约束和一个条件语句的全称量化等价。比如: ∀x<0(x2>0)≡∀x(x<0→x2>0)\forall x < 0 (x^{2} > 0) \equiv \forall x (x < 0 \to x^{2} > 0)∀x<0(x2>0)≡∀x(x<0→x2>0)。

而存在量化的约束和一个合取的存在量化等价。比如∃z>0(z2=2)≡∃z(z>0∧z2=2)\exist z > 0 (z^{2} = 2) \equiv \exist z(z > 0 \wedge z^{2} = 2)∃z>0(z2=2)≡∃z(z>0∧z2=2)。

Precedence of Quantifies

∀\forall∀和∃\exist∃比命题演算中的所有逻辑符号的优先级还要高。比如∀xP(x)∨Q(x)\forall x P(x) \vee Q(x)∀xP(x)∨Q(x)是∀xP(x)\forall x P(x)∀xP(x)和Q(x)Q(x)Q(x)的合取，即∀xP(x)∨Q(x)≡(∀xP(x))∨Q(x)\forall x P(x) \vee Q(x) \equiv (\forall x P(x)) \vee Q(x)∀xP(x)∨Q(x)≡(∀xP(x))∨Q(x)。

Binding Variable

当量词作用于一个变量xxx上，我们就说变量的这次出现是绑定的。当变量没有被量词绑定或者变量被赋予特定的值的时候，我们就说变量的这次出现时自由的。

逻辑表达式中应用量词的部分称为这个量词的作用域。因此，如果一个变量在指定该公式的所有量词的作用域之外，那么它就是自由的。

例如，∃x(x+y=1)\exist x (x + y = 1)∃x(x+y=1)，变量xxx被存在量化∃x\exist x∃x所绑定，但是变量yyy是自由的，因为这里没有量词绑定在该变量上，也没有给该变量赋予值。

在∃x(P(x)∧Q(x))∨∀xR(x)\exist x (P(x) \wedge Q(x)) \vee \forall x R(x)∃x(P(x)∧Q(x))∨∀xR(x)中，所有的变量都是绑定的。第一个量词∃x\exist x∃x的作用域，是表达式P(x)∧Q(x)P(x) \wedge Q(x)P(x)∧Q(x)，因为∃x\exist x∃x只作用在P(x)∧Q(x)P(x) \wedge Q(x)P(x)∧Q(x)上。同样的，第二个量词∀x\forall x∀x的作用域是表达式R(x)R(x)R(x)。因为两个量词的作用域没有重叠，所以我们可以用两个不同的变量符号去重写该表达式为∃x(P(x)∧Q(x))∨∀yR(y)\exist x (P(x) \wedge Q(x)) \vee \forall y R(y)∃x(P(x)∧Q(x))∨∀yR(y).

注意，相同的字母经常用于表示作用域不重叠的不同的量词绑定的变量。

Logical Equivalences Involving Quantifiers

定义3 \quad 涉及到谓词和量词的命题的逻辑等价当且仅当它们具有相同的真值，无论哪一个谓词被带入这些语句，也无论这些命题函数的变量的论域是什么。 我们用S≡TS \equiv TS≡T来表示两个包含谓词和量词的句子是逻辑等价的。

比如，证明∀x(P(x)∧Q(x))≡∀xP(x)∧∀xQ(x)\forall x (P(x) \wedge Q(x)) \equiv \forall x P(x) \wedge \forall x Q(x)∀x(P(x)∧Q(x))≡∀xP(x)∧∀xQ(x)，其中变量的论域相同。

证明: 要证明它们等价，我们必须证明它们具有相同的真值，无论谓词PPP和QQQ是什么，也无论变量的论域是什么。 假定我们有特定的谓词PPP和QQQ和相同的论域。

假设∀x(P(x)∧Q(x))\forall x (P(x) \wedge Q(x))∀x(P(x)∧Q(x))为真，这意味着如果aaa是论域的一个元素，那么P(a)∧Q(a)P(a) \wedge Q(a)P(a)∧Q(a)为真，因此P(a)P(a)P(a)为真，而且Q(a)Q(a)Q(a)也为真。因为P(a)P(a)P(a)为真，Q(a)Q(a)Q(a)为真是对于论域中的每一个元素，所以我们可以得出∀xP(x)\forall x P(x)∀xP(x)和∀xQ(x)\forall x Q(x)∀xQ(x)都为真。

假设∀xP(x)∧∀xQ(x)\forall x P(x) \wedge \forall x Q(x)∀xP(x)∧∀xQ(x)都为真，那么∀xP(x)\forall x P(x)∀xP(x)为真，且∀xQ(x)\forall x Q(x)∀xQ(x)为真。因此，如果aaa是论域的一个元素，那么P(a)P(a)P(a)为真，Q(a)Q(a)Q(a)也为真，所以P(a)∧Q(a)P(a) \wedge Q(a)P(a)∧Q(a)也为真。而aaa是论域中的任意一个元素，所以∀x(P(x)∧Q(x))\forall x (P(x) \wedge Q(x))∀x(P(x)∧Q(x))也为真。

因此，我么可以得出∀x(P(x)∧Q(x))≡∀xP(x)∧∀xQ(x)\forall x (P(x) \wedge Q(x)) \equiv \forall x P(x) \wedge \forall x Q(x)∀x(P(x)∧Q(x))≡∀xP(x)∧∀xQ(x)。

同理，我们也可以得出∃x(P(x)∨Q(x))≡∃xP(x)∨∃xQ(x)\exist x (P(x) \vee Q(x)) \equiv \exist x P(x) \vee \exist x Q(x)∃x(P(x)∨Q(x))≡∃xP(x)∨∃xQ(x)。

注意，全称量词无法对析取联结词分配，存在量词无法对合取联结词分配，即

∀x(P(x)∨Q(x))≢∀xP(x)∨∀xQ(x)\forall x(P(x) \vee Q (x)) \not\equiv \forall x P(x) \vee \forall x Q(x) ∀x(P(x)∨Q(x))​≡∀xP(x)∨∀xQ(x)

和

∃x(P(x)∧Q(x))≢∃xP(x)∧∃xQ(x)\exist x (P(x) \wedge Q(x)) \not \equiv \exist x P(x) \wedge \exist x Q(x) ∃x(P(x)∧Q(x))​≡∃xP(x)∧∃xQ(x)
