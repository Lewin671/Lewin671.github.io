Windows10的Office激活

Posted on 2020-02-04 Edited on 2020-10-17 In Configuration
  1. 先下载Office Tool Plus
  2. 若电脑没有安装office，则先部署office。这里选择office2019专业增强版。
  3. 安装对应的office许可证。
  4. 安装相应版本office的密钥。具体查询地址: http://www.ylmfwin100.net/ylmf/15106.html，对应的专业增强版的密钥是: NMMKJ-6RK4F-KMJVX-8D9MJ-6MWKP。
  5. 设置KMS服务器，可以使用kms.03k.org。
  6. 最后点击激活即可。

注意: 如果安装失败，则先卸载所有秘钥和重新删除许可证再重试即可。
