Java易错知识

Posted on 2020-09-14 In Programming Language , Java

这里记录一些Java中容易误解的一些知识点.

  1. Java的跨平台特性是因为JVM的存在，它可以执行.class字节码文件，而不是因为它的源代码可以在多个平台运行.
  2. 接口中的默认方法是abstract public,变量默认是用public final static修饰的。所以，接口中的变量是静态常量，常量不管在接口中还是类中必须声明时初始化。
