齐次线性方程组的解空间

设齐次线性方程组

AX=O(1)AX = O \tag{1} AX=O(1)

其中

A=[aij]m×n,X=(x1,x2,⋯ ,xn)T,O=(0,0,⋯ ,0)1×mTA = [a_{ij}]_{m \times n}, X = (x_{1},x_{2},\cdots,x_{n})^{T},O = (0,0,\cdots,0)_{1 \times m}^{T} A=[aij​]m×n​,X=(x1​,x2​,⋯,xn​)T,O=(0,0,⋯,0)1×mT​

定理1: \quad 齐次线性方程组(1)(1)(1)有非零解的充分必要条件是：系数矩阵AAA的秩r<nr<nr<n; 而只有零解的充分必要条件为AAA的秩为r=nr = nr=n.

证明:

把方程组(1)(1)(1)写成下面这种形式:

x1α1+x2α2+⋯+xnαn=0(2)x_{1} \alpha_{1} + x_{2} \alpha_{2} + \cdots + x_{n} \alpha_{n} = 0 \tag{2} x1​α1​+x2​α2​+⋯+xn​αn​=0(2)

其中α1,α2,⋯ ,αn\alpha_{1},\alpha_{2},\cdots,\alpha_{n}α1​,α2​,⋯,αn​为矩阵AAA的列向量，x=(x1,x2,⋯ ,xn)Tx = (x_{1},x_{2},\cdots,x_{n})^{T}x=(x1​,x2​,⋯,xn​)T.

由于r(A)<nr(A) < nr(A)<n，于是α1,α2,⋯ ,αn\alpha_{1},\alpha_{2},\cdots,\alpha_{n}α1​,α2​,⋯,αn​线性相关，即存在不全为零的数使得成立，即存在x≠0x \not={0}x​=0使得(1)(1)(1)成立。

反之，若方程组(1)(1)(1)存在非零解xxx，则由(2)(2)(2)知道α1,α2,⋯ ,αn\alpha_{1},\alpha_{2},\cdots,\alpha_{n}α1​,α2​,⋯,αn​线性相关，从而r(A)<nr(A) < nr(A)<n.

对第一个命题取逆否命题，得到： 齐次线性方程组没有非零解的充分必要条件是：系数矩阵AAA的秩r≥nr \ge nr≥n.

又因为r≤nr \le nr≤n，所以只有零解的充分必要条件是r=nr = nr=n.

定理2:\quad 设ξ\xiξ和η\etaη是线性方程组的两个解向量，则k1ξ+k2ηk_{1} \xi + k_{2} \etak1​ξ+k2​η也是线性方程组的解向量。 从而，齐次线性方程组的解构成一个向量空间，我们把这个空间叫做齐次线性方程组的解空间。

定义1: \quad 一齐次线性方程组若有非零解，则称其一组解向量ξ1,ξ2,⋯ ,ξn\xi_{1},\xi_{2},\cdots,\xi_{n}ξ1​,ξ2​,⋯,ξn​为该齐次线性方程组的一个基础解系，如果:

  • ξ1,ξ2,⋯ ,ξn\xi_{1},\xi_{2},\cdots,\xi_{n}ξ1​,ξ2​,⋯,ξn​线性无关
  • 该齐次线性方程组的任一解向量均可由ξ1,ξ2,⋯ ,ξn\xi_{1},\xi_{2},\cdots,\xi_{n}ξ1​,ξ2​,⋯,ξn​线性表出。

定理3: \quad 设齐次线性方程组(1)(1)(1)的系数矩阵AAA的秩为r<nr < nr<n，则齐次线性方程组(1)(1)(1)的基础解系由n−rn-rn−r个向量组成，即解空间的维数为n−rn -rn−r.

注意，这个性质是齐次方程组的时候才有这个性质.

非齐次线性方程组解的讨论

设非齐次线性方程组为

AX=β(5)AX = \beta \tag{5} AX=β(5)

其中

A=[aij]m×n,X=(x1,x2,⋯ ,xn)T,β=(b1,b2,⋯ ,bn)A = [a_{ij}]_{m \times n}, X = (x_{1},x_{2},\cdots,x_{n})^{T},\beta = (b_{1},b_{2},\cdots,b_{n}) A=[aij​]m×n​,X=(x1​,x2​,⋯,xn​)T,β=(b1​,b2​,⋯,bn​)

定理4: \quad 设线性方程组(5)(5)(5)的系数矩阵的秩为rrr，增广矩阵的秩为r′r'r′.那么:

  • r≠r′r \not={r'}r​=r′，则方程组(5)(5)(5)无解.
  • r=r′=nr = r' = nr=r′=n，则方程组(5)(5)(5)有唯一解.
  • r=r′<nr = r' < nr=r′<n，则方程组(5)(5)(5)有无穷多解.

证明: 若r≠r′r\not={r'}r​=r′，则说明β\betaβ不能被α1,α2,⋯ ,αn\alpha_{1},\alpha_{2},\cdots,\alpha_{n}α1​,α2​,⋯,αn​线性表示，从而线性方程组

x1α1+⋯+xnαn=βx_{1} \alpha_{1} + \cdots + x_{n} \alpha_{n} = \beta x1​α1​+⋯+xn​αn​=β

无解.

同理，当r=r′r = r'r=r′时，方程组有解.

当r=r′=nr = r'= nr=r′=n时，α1,⋯ ,αn\alpha_{1},\cdots,\alpha_{n}α1​,⋯,αn​线性无关，所以β\betaβ可以被α1,⋯ ,αn\alpha_{1},\cdots,\alpha_{n}α1​,⋯,αn​唯一地线性表示，即x1,x2,⋯ ,xnx_{1},x_{2},\cdots,x_{n}x1​,x2​,⋯,xn​唯一.

当r=r′<nr = r' < nr=r′<n时，通过下面的定理555可以知道，非齐次线性方程组的解有无穷多个.
