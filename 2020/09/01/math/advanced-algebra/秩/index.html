向量组的秩和矩阵的秩

Posted on 2020-09-01 Edited on 2020-12-10 In Math , 高等代数

向量组的秩

极大线性无关组

在定义向量组的秩的之前，先介绍极大线性无关组。

定义1: \quad 称一组向量组α1,α2,⋯ ,αm\alpha_{1},\alpha_{2},\cdots,\alpha_{m}α1​,α2​,⋯,αm​的部分向量组αi1,αi2,⋯ ,αir(1≤i1≤i2≤⋯≤ir≤m)\alpha_{i_{1}},\alpha_{i_2},\cdots,\alpha_{i_{r}}(1 \le i_{1} \le i_{2} \le \cdots \le i_{r} \le m)αi1​​,αi2​​,⋯,αir​​(1≤i1​≤i2​≤⋯≤ir​≤m)为一极大线性无关组，如果

  • αi1,αi2,⋯ ,αir\alpha_{i_{1}},\alpha_{i_2},\cdots,\alpha_{i_{r}}αi1​​,αi2​​,⋯,αir​​线性无关.
  • ∀αj(j=1,2,⋯ ,m)\forall \alpha_{j}(j = 1,2,\cdots,m)∀αj​(j=1,2,⋯,m)都可以由αi1,αi2,⋯ ,αir\alpha_{i_{1}},\alpha_{i_2},\cdots,\alpha_{i_{r}}αi1​​,αi2​​,⋯,αir​​线性表出。

定理1: \quad一个向量组的任意两个极大无关组所含向量个数相等。

证明:

假设一个向量组的任意两个极大线性无关组分别为αi1,αi2,⋯ ,αis\alpha_{i_{1}},\alpha_{i_2},\cdots,\alpha_{i_{s}}αi1​​,αi2​​,⋯,αis​​和αj1,αj2,⋯ ,αjt\alpha_{j_{1}},\alpha_{j_2},\cdots,\alpha_{j_{t}}αj1​​,αj2​​,⋯,αjt​​，则根据极大线性无关组的定义，αi1,αi2,⋯ ,αis\alpha_{i_{1}},\alpha_{i_2},\cdots,\alpha_{i_{s}}αi1​​,αi2​​,⋯,αis​​线性无关，并且可以被另一极大线性无关组αj1,αj2,⋯ ,αjt\alpha_{j_{1}},\alpha_{j_2},\cdots,\alpha_{j_{t}}αj1​​,αj2​​,⋯,αjt​​所线性表示，根据替换定理，有s≤ts \le ts≤t.

同理，有t≤st \le st≤s，于是t=st = st=s.

或者： 两个向量组分别线性无关，并且等价，因此所含的向量个数相等。

向量组的秩

因为向量组的极大线性无关组的向量个数是固定的，因此我们把它定义为向量组的秩。

定义2: \quad 一个向量组的任一极大线性无关组所含的向量个数称为此向量组的秩。

定理2: \quad 设向量组α1,α2,⋯ ,αm\alpha_{1},\alpha_{2},\cdots,\alpha_{m}α1​,α2​,⋯,αm​可由向量组β1,β2,⋯ ,βn\beta_{1},\beta_{2},\cdots,\beta_{n}β1​,β2​,⋯,βn​线性表示，则向量组α1,α2,⋯ ,αm\alpha_{1},\alpha_{2},\cdots,\alpha_{m}α1​,α2​,⋯,αm​的秩不大于向量组β1,β2,⋯ ,βn\beta_{1},\beta_{2},\cdots,\beta_{n}β1​,β2​,⋯,βn​的秩。

证明:

不妨设α1,α2,⋯ ,αm\alpha_{1},\alpha_{2},\cdots,\alpha_{m}α1​,α2​,⋯,αm​的秩为sss，其中一个极大线性无关组为αi1,αi2,⋯ ,αis\alpha_{i_{1}},\alpha_{i_{2}},\cdots,\alpha_{i_{s}}αi1​​,αi2​​,⋯,αis​​，β1,β2,⋯ ,βn\beta_{1},\beta_{2},\cdots,\beta_{n}β1​,β2​,⋯,βn​的秩为ttt，其中一个极大线性无关组为βj1,βj2,⋯ ,βjt\beta_{j_1},\beta_{j_2},\cdots,\beta_{j_t}βj1​​,βj2​​,⋯,βjt​​.

根据线性表示的传递性可知，αi1,αi2,⋯ ,αis\alpha_{i_{1}},\alpha_{i_{2}},\cdots,\alpha_{i_{s}}αi1​​,αi2​​,⋯,αis​​可以被βj1,βj2,⋯ ,βjt\beta_{j_1},\beta_{j_2},\cdots,\beta_{j_t}βj1​​,βj2​​,⋯,βjt​​线性表示，并且αi1,αi2,⋯ ,αis\alpha_{i_{1}},\alpha_{i_{2}},\cdots,\alpha_{i_{s}}αi1​​,αi2​​,⋯,αis​​线性无关，于是s≤ts \le ts≤t.

计算极大无关组

我们一般是通过对向量组进行初等行变换来计算矩阵向量组(α1,α2,⋯ ,αn)(\alpha_{1},\alpha_{2},\cdots,\alpha_{n})(α1​,α2​,⋯,αn​)的极大无关组。

原理如下:

假设AAA是一个可逆矩阵，则

A(α1,α2,⋯ ,αn)=(β1,β2,⋯ ,βn)A(\alpha_{1},\alpha_{2},\cdots,\alpha_{n}) = (\beta_{1},\beta_{2},\cdots,\beta_{n}) A(α1​,α2​,⋯,αn​)=(β1​,β2​,⋯,βn​)

表示(β1,β2,⋯ ,βn)(\beta_{1},\beta_{2},\cdots,\beta_{n})(β1​,β2​,⋯,βn​)可以被(α1,α2,⋯ ,αn)(\alpha_{1},\alpha_{2},\cdots,\alpha_{n})(α1​,α2​,⋯,αn​)线性表示. 同时，将方程两边左乘A−1A^{-1}A−1，则得到新的方程

(α1,α2,⋯ ,αn)=A−1(β1,β2,⋯ ,βn)(\alpha_{1},\alpha_{2},\cdots,\alpha_{n}) = A^{-1}(\beta_{1},\beta_{2},\cdots,\beta_{n}) (α1​,α2​,⋯,αn​)=A−1(β1​,β2​,⋯,βn​)

这表示(α1,α2,⋯ ,αn)(\alpha_{1},\alpha_{2},\cdots,\alpha_{n})(α1​,α2​,⋯,αn​)也可以被(β1,β2,⋯ ,βn)(\beta_{1},\beta_{2},\cdots,\beta_{n})(β1​,β2​,⋯,βn​)线性表示。

根据定理2，(α1,α2,⋯ ,αn)(\alpha_{1},\alpha_{2},\cdots,\alpha_{n})(α1​,α2​,⋯,αn​)和(β1,β2,⋯ ,βn)(\beta_{1},\beta_{2},\cdots,\beta_{n})(β1​,β2​,⋯,βn​)的秩相等，即极大无关组的向量个数相同。

这说明，初等行变换不改变向量组的秩。

不妨假设(α1,α2,⋯ ,αn)(\alpha_{1},\alpha_{2},\cdots,\alpha_{n})(α1​,α2​,⋯,αn​)的前mmm个向量为极大无关组，有

k1α1+k2α2+⋯+kmαm=0k_{1} \alpha_{1} + k_{2} \alpha_{2} + \cdots + k_{m} \alpha_{m} = 0 k1​α1​+k2​α2​+⋯+km​αm​=0

当且仅当k1=k2=⋯=km=0k_{1} = k_{2} = \cdots = k_{m} = 0k1​=k2​=⋯=km​=0

对这个方程方程两边同时乘以可逆矩阵AAA，不影响kik_{i}ki​的取值，因此

k1Aα1+k2Aα2+⋯+kmAαm=0k_{1}A\alpha_{1} + k_{2} A \alpha_{2} + \cdots + k_{m} A\alpha_{m} = 0 k1​Aα1​+k2​Aα2​+⋯+km​Aαm​=0

仍当且仅当k1=k2=⋯=km=0k_{1} = k_{2} = \cdots = k_{m} = 0k1​=k2​=⋯=km​=0.

即

k1β1+k2β2+⋯+kmβm=0k_{1} \beta_{1} + k_{2} \beta_{2} + \cdots + k_{m} \beta_{m} = 0 k1​β1​+k2​β2​+⋯+km​βm​=0

当且仅当k1=k2=⋯=km=0k_{1} = k_{2} = \cdots = k_{m} = 0k1​=k2​=⋯=km​=0.

这说明: 经过初等行变换之后，得到的β\betaβ的向量组β1,β2,⋯ ,βm\beta_{1},\beta_{2},\cdots,\beta_{m}β1​,β2​,⋯,βm​所对应的α1,α2,⋯ ,αm\alpha_{1},\alpha_{2},\cdots,\alpha_{m}α1​,α2​,⋯,αm​的线性相关性相同。

所以，只需要找到变换后的极大无关组所对应的原来的列，这些列就是原向量组的极大无关组。
