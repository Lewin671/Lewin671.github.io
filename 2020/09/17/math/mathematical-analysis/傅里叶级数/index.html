三角级数

最简单的周期函数，可以用正弦函数

y=Asin⁡(ωx+φ)(1)y = A \sin(\omega x + \varphi) \tag{1} y=Asin(ωx+φ)(1)

来表示.

由(1)(1)(1)表示的周期运动也叫做简谐运动. 其中AAA为振幅，φ\varphiφ为初相角，ω\omegaω为角频率，于是简谐运动yyy的周期是T=2πωT = \dfrac{2 \pi}{\omega}T=ω2π​.

对于复杂的周期运动，则常常是几个简谐震动

yk=Aksin⁡(kωx+φk),k=1,2,⋯ ,ny_{k} = A_{k} \sin(k\omega x + \varphi_{k}), k = 1,2,\cdots,n yk​=Ak​sin(kωx+φk​),k=1,2,⋯,n

的叠加

y=∑k=1nyk=∑k=1nAksin⁡(kωx+φk)(2)y = \sum_{k = 1}^{n} y_{k} = \sum_{k = 1}^{n} A_{k} \sin(k \omega x + \varphi_{k}) \tag{2} y=k=1∑n​yk​=k=1∑n​Ak​sin(kωx+φk​)(2)

由于yky_{k}yk​的最小正周期为Tk\dfrac{T}{k}kT​，所以TTT是yyy的一个周期.

对无穷多个简谐运动进行叠加就得到函数项级数

A0+∑n=1∞Ansin⁡(nωx+φn)(3)A_{0} + \sum_{n = 1}^{\infty} A_{n} \sin(n \omega x + \varphi_{n}) \tag{3} A0​+n=1∑∞​An​sin(nωx+φn​)(3)

若级数(3)(3)(3)收敛，则它描述的是更为一般的周期运动现象。

对于级数(3)(3)(3)，我们只需要讨论w=1w = 1w=1的情形，如果w≠1w \not={1}w​=1，则用t=wxt = w xt=wx进行代换.

通过三角函数的公式，把级数(3)(3)(3)展开，得到

A0+∑n=1∞(Ancos⁡φnsin⁡nx+Ansin⁡φncos⁡nx)A_{0} + \sum_{n = 1}^{\infty} (A_{n}\cos \varphi_{n} \sin n x + A_{n} \sin \varphi_{n} \cos n x) A0​+n=1∑∞​(An​cosφn​sinnx+An​sinφn​cosnx)

为了后面的公式的统一，我们记

A0=a02,Ancos⁡nx=an,Ansin⁡nx=bnA_{0} = \dfrac{a_{0}}{2}, A_{n} \cos n x = a_{n}, A_{n} \sin nx = b_{n} A0​=2a0​​,An​cosnx=an​,An​sinnx=bn​

则级数(3)(3)(3)可以写成

a02+∑n=1∞(ancos⁡nx+bnsin⁡nx)(4)\dfrac{a_{0}}{2} + \sum_{n = 1}^{\infty} (a_{n} \cos n x + b_{n} \sin n x) \tag{4} 2a0​​+n=1∑∞​(an​cosnx+bn​sinnx)(4)

它是由三角函数列

1,cos⁡x,sin⁡x,cos⁡2x,sin⁡2x,⋯ ,cos⁡nx,sin⁡nx,⋯(5)1, \cos x, \sin x, \cos 2x , \sin 2x, \cdots, \cos nx ,\sin nx,\cdots \tag{5} 1,cosx,sinx,cos2x,sin2x,⋯,cosnx,sinnx,⋯(5)

所产生的一般形式的三角级数.

定理1:\quad 若级数

∣a0∣2+∑n=1∞(∣an∣+∣bn∣)\dfrac{|a_{0}|}{2} + \sum_{n = 1}^{\infty}(|a_{n}| + |b_{n}|) 2∣a0​∣​+n=1∑∞​(∣an​∣+∣bn​∣)

收敛，则级数(4)(4)(4)在整个数轴上绝对一致收敛.

证明:

由于

∣ancos⁡nx+bnsin⁡nx∣≤∣an∣+∣bn∣|a_{n} \cos n x + b_{n} \sin nx | \le |a_{n}| + |b_{n}| ∣an​cosnx+bn​sinnx∣≤∣an​∣+∣bn​∣

根据优级数判别法可知，级数(4)(4)(4)绝对一致收敛.

三角函数列的性质： 在三角函数系(5)(5)(5)中，任何两个不相同的函数的乘积在[−π,π][-\pi,\pi][−π,π]的积分都等于0，即

∫−ππsin⁡nxcos⁡mxdx=0(7)\int_{-\pi}^{\pi} \sin n x \cos m x \mathrm{d}x= 0 \tag{7} ∫−ππ​sinnxcosmxdx=0(7)

∫−ππcos⁡nxcos⁡mxdx=0(m≠n)(9)\int_{-\pi}^{\pi} \cos nx \cos mx \mathrm{d}x = 0(m \not={n}) \tag{9} ∫−ππ​cosnxcosmxdx=0(m​=n)(9)

而(5)(5)(5)中任何一个函数的平方在[−π,π][-\pi,\pi][−π,π]上的积分都不等于0，即

∫−ππcos⁡2nxdx=∫−ππsin⁡2nxdx=π(10)\int_{-\pi}^{\pi} \cos^2 nx \mathrm{d}x = \int_{-\pi}^{\pi} \sin^{2} nx \mathrm{d}x = \pi \tag{10} ∫−ππ​cos2nxdx=∫−ππ​sin2nxdx=π(10)
