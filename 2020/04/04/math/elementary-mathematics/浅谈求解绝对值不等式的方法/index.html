浅谈求解绝对值不等式的方法

Posted on 2020-04-04 Edited on 2020-05-11 In Math , 初等数学

介绍

在我们高中的求解绝对值不等式的时候，往往只涉及到一个绝对值的不等式，比如

∣x−a∣<b|x - a| < b ∣x−a∣<b

这种不等式很容易通过去绝对值，得到

−b<x−a<b-b < x - a < b −b<x−a<b

求解。

对于两个绝对值的不等式，形如

∣x−a∣+∣x−b∣<c|x - a| + |x - b | < c ∣x−a∣+∣x−b∣<c

我们往往比较难以下手，这里我就介绍一种方法来解这种不等式。

方法

这里主要通过“平方”的方法去掉绝对值，主要的依据是

∣x−a∣<c→(x−a)2<c2|x - a| < c \to (x-a)^2 < c^2 ∣x−a∣<c→(x−a)2<c2

但是，需要注意的是，通过(x−a)2<c2(x-a)^2 < c^2(x−a)2<c2求解出来的解和∣x−a∣<c|x-a| < c∣x−a∣<c的解往往不是等价的。因为

(x−a)2<c2→(∣x−a∣<c)∨(∣x−a∣<−c)(x-a)^2 < c^2 \to (|x-a| < c) \vee(|x-a| < -c) (x−a)2<c2→(∣x−a∣<c)∨(∣x−a∣<−c)

所以，我们往往会构造出c>0c>0c>0来使得(x−a)2<c2(x-a)^2 < c^2(x−a)2<c2和∣x−a∣<c|x-a| < c∣x−a∣<c的解等价。即

(x−a)2<c2∧c>0→∣x−a∣<c(x-a)^2 < c^2 \wedge c>0 \to |x-a| < c (x−a)2<c2∧c>0→∣x−a∣<c

如果无法构造出c>0c>0c>0，则我们可以通过后验来排除解。

例如:

  1. 求解

    ∣x+2∣+∣x−2∣≤12|x+2| + |x-2| \le 12 ∣x+2∣+∣x−2∣≤12

    解:

    通过移项，得到

    ∣x+2∣≤12−∣x−2∣|x + 2| \le 12 - |x-2| ∣x+2∣≤12−∣x−2∣

    对上式两边进行平方:

    x2+4x+4≤144−24∣x−2∣+(x2−4x+4)x^2 + 4x + 4 \le 144 - 24 |x-2| + (x^2 - 4x + 4) x2+4x+4≤144−24∣x−2∣+(x2−4x+4)

    整理得到:

    3∣x−2∣≤18−x3|x-2| \le 18 -x 3∣x−2∣≤18−x

    再对两边平方

    9(x2−4x+4)≤324−36x+x29(x^2 - 4x + 4) \le 324 - 36x + x^2 9(x2−4x+4)≤324−36x+x2

    整理得到:

    x2−36≤0x^2 - 36 \le 0 x2−36≤0

    所以

    −6≤x≤6-6 \le x \le 6 −6≤x≤6

    下面检验这个解，因为∣x∣≤6|x| \le 6∣x∣≤6可以使得12−∣x−2∣>012 - |x-2| >012−∣x−2∣>0且18−x>018 -x > 018−x>0，所以解是没问题的。

  2. 求解∣x+2∣−∣x∣>1|x+2| - |x| > 1∣x+2∣−∣x∣>1

    解:

    ∣x+2∣>1+∣x∣|x+2| > 1 + |x| ∣x+2∣>1+∣x∣

    两边平方（注意，等式两边都大于0）

    x2+4x+4>1+2∣x∣+x2x^2 + 4x + 4 > 1 + 2|x| + x^2 x2+4x+4>1+2∣x∣+x2

    整理得到

    2∣x∣<4x+32|x| < 4x + 3 2∣x∣<4x+3

    两边再进行平方(注意，这里要确保4x+3>04x + 3 > 04x+3>0)，得到

    4x2+8x+3>04x^2 + 8x + 3 > 0 4x2+8x+3>0

    解得

    x>−1/2x > -1/2 x>−1/2

    或

    x<−3/2x < -3/2 x<−3/2

    而后者使得4x+3<04x + 3 < 04x+3<0，所以舍去，所以不等式的解为

    x>−1/2x > -1/2 x>−1/2
