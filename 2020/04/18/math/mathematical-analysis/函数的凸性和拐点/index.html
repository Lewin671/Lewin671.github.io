函数的凸性和拐点

Posted on 2020-04-18 Edited on 2020-10-20 In Math , 数学分析

我们把曲线上任意两点的弧段总在这两点的下方的函数称为凸函数，反之，若函数曲线的任意两点的弧段总在这两点的上方，则称该函数为凹函数。

凹点和凸点

定义1： \quad 设fff为定义在区间III上的函数，若对III上的任意两点x1,x2x_{1},x_{2}x1​,x2​和任意实数λ∈(0,1)\lambda \in (0,1)λ∈(0,1)，总有

f(λx1+(1−λ)x2)≤λf(x1)+(1−λ)f(x2)(1)f(\lambda x_{1} + (1-\lambda) x_{2}) \le \lambda f(x_{1}) + (1-\lambda)f(x_{2}) \tag{1} f(λx1​+(1−λ)x2​)≤λf(x1​)+(1−λ)f(x2​)(1)

则称fff为III上的凹函数. 反之，如果总有

f(λx1+(1−λ)x2)≥λf(x1)+(1−λ)f(x2)(2)f(\lambda x_{1} + (1-\lambda)x_{2}) \ge \lambda f(x_{1}) + (1-\lambda)f(x_{2}) \tag{2} f(λx1​+(1−λ)x2​)≥λf(x1​)+(1−λ)f(x2​)(2)

则称fff为III上的凸函数。

注: 华东师范教材的定义与这里的定义相反。

若把(1),(2)(1),(2)(1),(2)上的不等式改为严格不等式，则相应的函数称为严格凸函数和严格凹函数。

注: (1),(2)(1),(2)(1),(2)式的几何意义可能不是很明显。我这里就拿凸函数的定义为例子，且不妨假设x1<x2x_{1} < x_{2}x1​<x2​. 那么任意x∈[x1,x2]x \in [x_{1},x_{2}]x∈[x1​,x2​]都可以被λx1+(1−λ)x2\lambda x_{1} + (1-\lambda) x_{2}λx1​+(1−λ)x2​表示.

因此对于(1)(1)(1)式左端表示fff在λx1+(1−λ)x2\lambda x_{1} + (1-\lambda) x_{2}λx1​+(1−λ)x2​的函数值，而右端表示割线方程x−x1x2−x1=y−f(x1)f(x2)−f(x1)\dfrac{x - x_{1}}{x_{2} - x_{1}} = \dfrac{y-f(x_{1})}{f(x_{2}) - f(x_{1})}x2​−x1​x−x1​​=f(x2​)−f(x1​)y−f(x1​)​在λx1+(1−λ)x2\lambda x_{1} + (1-\lambda) x_{2}λx1​+(1−λ)x2​的函数值。所以整个不等式表示，对于任意x∈(x1,x2)x \in (x_{1},x_{2})x∈(x1​,x2​)，任意两点的曲线弧总在这两点所在的割线的下方。

定理1: \quad 设fff为区间III上的二阶可导函数，则在III上fff为凸函数的充要条件是

f′′(x)≤0(5)f''(x) \le 0 \tag{5} f′′(x)≤0(5)

在III上fff为凹函数的充要条件为

f′′(x)≥0(6)f''(x) \ge 0 \tag{6} f′′(x)≥0(6)

拐点

定义2: \quad 设曲线y=f(x)y=f(x)y=f(x)在点(x0,f(x0))(x_{0},f(x_{0}))(x0​,f(x0​))处有穿过曲线的切线. 且在切点近旁，曲线在切线的两侧分别是严格凸和严格凹的，这是称点(x0,f(x0))(x_{0},f(x_{0}))(x0​,f(x0​))为曲线y=f(x)y = f(x)y=f(x)的拐点。

注意: 和极值点、零点中“点”的定义不同，拐点需要带上横坐标和纵坐标。

定理2: \quad 若fff在x0x_{0}x0​二阶可导，则(x0,f(x0))(x_{0},f(x_{0}))(x0​,f(x0​))为曲线y=f(x)y = f(x)y=f(x)的拐点的必要条件是f′′(x0)=0f''(x_{0}) = 0f′′(x0​)=0.

定理4(判断拐点的第二充分条件): \quad 设f(x)f(x)f(x)在x=x0x = x_{0}x=x0​的某领域内三阶可导，且f′′(x0)=0,f′′′(x0)≠0f''(x_{0}) = 0, f'''(x_{0}) \not={0}f′′(x0​)=0,f′′′(x0​)​=0，则(x0,f(x0))(x_{0},f(x_{0}))(x0​,f(x0​))为拐点。
