三角函数

三角函数（英语：Trigonometric functions）是数学中常见的一类关于角度的函数。三角函数将直角三角形的内角和它的两个边的比值相关联，也可以等价地用与单位圆有关的各种线段的长度来定义。三角函数在研究三角形和圆等几何形状的性质时有重要作用，也是研究周期性现象的基础数学工具。在数学分析中，三角函数也被定义为无穷级数或特定微分方程的解，允许它们的取值扩展到任意实数值，甚至是复数值.

ssh免密登录

Posted on 2019-11-04 Edited on 2020-04-27 In Configuration

与密码验证相对的，是公钥验证。也就是说，要实现免密码登入，首先要设置 SSH 钥匙。

这段话告诉了我们，生成的公钥放在了 ~/.ssh/id_rsa.pub，私钥放在了 ~/.ssh/id_rsa。接下来，我们要让远程机器记住我们的公钥 。

下面提供两种方法:

  • ssh-copy-id user@remote -p port
  • 在没有 ssh-copy-id 的情况下（比如在 Windows 上），也是可以轻松做到这件事的。用命令的话也是一句话搞定 : ssh user@remote -p port 'mkdir -p .ssh && cat >> .ssh/authorized_keys' < ~/.ssh/id_rsa.pub

如果上面两种方法都无法免密登录，说明ssh出了一些问题。我们连接的时候可以通过添加-v选项查看调试信息。

比如出现了Ignored authorized keys: bad ownership or modes for file /home/***/.ssh/authorized_keys说明authorized_keys文件的权限有问题，这个时候我们把这个文件的权限设置为600即可。
