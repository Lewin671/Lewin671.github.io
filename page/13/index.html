其中，threading.Thread类的构造函数为：
threading.Thread(group=None,target=None,name=None,args=(),kwargs={},*,daemon=None)

参数说明：

  • group是以后的拓展（当ThreadGroup实现的时候)
  • target是回调目标用于被run()方法调用的，默认为None，表示没有任何东西被调用。
  • name是线程的名字，默认一个唯一的名字是这样构造的：Thread-N。
  • args是一个元组，表示target的参数，默认为()。
  • kwargs是一个target关键字字典，默认为{}
  • daemon是守护进程

启动线程start()

调用start()函数之后，线程会自动run方法。

join(timeout=None)

这个函数的作用是: wait until thread terminates。也就是说，会一直运行当前的线程，直到当前线程结束。我们可以使用is_alive()来判断进程是否结束。

name

线程的名字，可以直接通过点操作符(.)调用name属性。

线程同步

由于在多线程中，数据是共享的，所以就有可能多个线程同时去改变一个变量，这个有时候会出问题。比如：我们的计算机的操作就经常要求关闭文件后才能打开，这就是因为如果同时去修改一个文件，会造成最终的结果不确定或者出错，所以我们就需要去同步，或者说，需要把数据“锁”上。在Python中，我们可以使用Lock和RLock类，他们都有相同的方法：

  • acquire(blocking=true,timeout=-1)
  • release()

根据英文意思你也知道上面两个方法的作用，acquire是获得一把锁，release是释放锁（即解锁的意思）。

元字符

正则表达式语言由两种基本字符类型组成：原义（正常）文本字符和元字符。元字符使正则表达式具有处理能力。所谓元字符就是指那些在正则表达式中具有特殊意义的专用字符，可以用来规定其前导字符（即位于元字符前面的字符）在目标对象中的出现模式。
下面列出常用的元字符：

  • .: 匹配除换行以外的任意字符
  • \w: 匹配字母或数字或下划线或汉字
  • \s: 匹配任意的空白符
  • \d: 匹配数字
  • \b: 匹配单词的开始或结束
  • ^: 匹配字符串的开始
  • $: 匹配字符串的结束

Posted on 2018-11-29 Edited on 2020-04-27 In Programming Language , Python

基本文件操作

创建和打开文件

在Python中，想要操作文件需要先创建或者打开指定文件，可以通过open（）函数实现。open（）函数的语法格式如下：

file = open(filename[,mode[,buffering]])

其中[]里面的参数表示是可选参数。

  • filename 表示文件路径
  • mode表示文件打开的方式
  • buffering: 用于指定读写文件的缓冲模式。值为0表示不缓存，值为1表示缓存，如果大于1表示缓冲区的大小。

打开一个不存在的文件时先创建该文件

在默认情况下，使用open打开一个不存在的文件会抛出异常。解决的方法有两种：

  • 自己手动在该路径建立一个文件
  • 打开文件的的时候赋予写权限，比如：w,w+,a,a+等等。这样，当要打开一个不存在的文件时，就可以创建一个新的文件了。

这篇文章主要介绍python的一些语法特点和习惯，包括注释，

注释

注释的作用就是对代码的解释和说明，让他人了解代码实现的功能，从而帮助程序猿更好地阅读代码。注释的内容python将不会执行，因此也可以用于临时不想执行的代码。

单行注释

在python中，使用#作为单行注释的符号。从符号#开始直到换行为止，#所有的内容都作为注释的内容，并被编译器忽略。
