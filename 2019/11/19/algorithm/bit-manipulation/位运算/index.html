异或(exclusive or)

异或是一种二元运算，当两两数值相同时为否，而数值相同时为真, 记做p ^ q或者p⊕qp \oplus qp⊕q，其中p和q可以是数字、命题等等.

异或运算的真值表

pqp ^ q
TTF
TFT
FTT
FFF

异或的性质

  1. 交换律. p ^ q = q ^ p.
  2. 结合律. (p ^ q) ^ r = p ^ (q ^ r).
  3. 恒等律. p ^ 0 = p.
  4. 归零律. p ^ p = 0.

异或的应用

  1. Given an array containing n distinct numbers taken from 0, 1, 2, …, n, find the one that is missing from the array.

    Example 1:

    解: 假设序列为{an}\{a_n\}{an​}, 其中的missing number是jjj,那么下标是{0,1,⋯ ,n−1}\{0,1,\cdots,n-1\}{0,1,⋯,n−1}，那么元素和下标的异或为

    res=(0⊕a0)⊕(1⊕a1)⊕⋯⊕((n−1)⊕an−1)=(0⊕0)⊕(1⊕1)⊕⋯⊕((j−1)⊕(j−1))⊕((j+1)⊕(j+1))⋯((n−1)⊕(n−1))⊕(n⊕j)=n⊕j\begin{aligned} res & = (0 \oplus a_0) \oplus (1 \oplus a_1) \oplus \cdots \oplus((n-1) \oplus a_{n-1}) \\ & = (0 \oplus 0) \oplus (1 \oplus 1) \oplus \cdots \oplus ((j-1) \oplus (j-1)) \\ & \oplus ((j+1) \oplus (j+1)) \cdots ((n-1) \oplus (n-1)) \oplus (n \oplus j) \\ & = n \oplus j \end{aligned}res​=(0⊕a0​)⊕(1⊕a1​)⊕⋯⊕((n−1)⊕an−1​)=(0⊕0)⊕(1⊕1)⊕⋯⊕((j−1)⊕(j−1))⊕((j+1)⊕(j+1))⋯((n−1)⊕(n−1))⊕(n⊕j)=n⊕j​

    注： 上面的第二个等式是根据异或的交换律和结合律得到的.
    所以n⊕res=jn \oplus res = jn⊕res=j,这样就可以得到缺失值了.

    所以，如果把问题稍微改一下，改成多了一个在0到n内的数字，让你求这个数字是什么，原理也是同样的.

  2. 交换两个数.

    证明:
    假设第一行的第一个aaa是a′a'a′，第二行的第一个bbb是b′b'b′，第三行的第一个aaa是a′′a''a′′，那么

    a′=a⊕bb′=a′⊕b=(a⊕b)⊕b=aa′′=a′⊕b′=(a⊕b)⊕a=b\begin{aligned} & a' = a \oplus b \\ & b' = a' \oplus b = (a \oplus b) \oplus b = a \\ & a'' = a' \oplus b'= (a \oplus b) \oplus a = b \end{aligned}​a′=a⊕bb′=a′⊕b=(a⊕b)⊕b=aa′′=a′⊕b′=(a⊕b)⊕a=b​
