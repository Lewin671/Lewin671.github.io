Visual Studio Code插件推荐

Posted on 2019-11-07 Edited on 2020-04-27 In Configuration

为了拥有更好的Coding体验，这里推荐一些Visual Studio Code的一些插件工具。

  • Bracket Pair Colorizer: 让不同层次的括号有不同的颜色。
  • C++ Algorithm Mnemonics： c++提示工具.
  • Markdown All in One. Markdown提示工具，支持mathjax或者katex
  • Markdown Preview Enhanced. Markdown预览工具.
  • Markdownlint. Markdown提示工具.
  • Path intellisense. 路径提示工具
  • Remote-Containers. 远程Visual Studio Code的工具.
  • Code Blue主题
  • Sync Setting. 配置文件同步
