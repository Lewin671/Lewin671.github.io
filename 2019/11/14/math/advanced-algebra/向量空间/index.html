向量空间

定义1 \quad 设VVV是一个非空集合，FFF是一个数域. VVV上定义了一个加法的代数运算，即对于任意的α,β∈V\alpha,\beta \in Vα,β∈V，存在唯一的α+β∈V\alpha + \beta \in Vα+β∈V，以及定义了一个数量乘法，使得对于任意的a∈Fa \in Fa∈F和α∈V\alpha \in Vα∈V，存在唯一的kα∈Vk \alpha \in Vkα∈V. 如果还满足如下的8条算律，称VVV为数域FFF上的一个向量空间，并称VVV中的元素为向量.

  1. α+β=β+α\alpha + \beta = \beta + \alphaα+β=β+α
  2. (α+β)+γ=α+(β+γ)(\alpha + \beta) + \gamma = \alpha + (\beta + \gamma)(α+β)+γ=α+(β+γ)
  3. ∃0∈V\exist 0 \in V∃0∈V,使得α+0=α\alpha + 0 = \alphaα+0=α
  4. ∀α∈V\forall \alpha \in V∀α∈V,∃−α∈V\exist -\alpha \in V∃−α∈V,使得α+(−α)=0\alpha + (-\alpha) = 0α+(−α)=0
  5. 1α=α1\alpha = \alpha1α=α
  6. (kl)α=k(lα)(kl)\alpha = k(l\alpha)(kl)α=k(lα)
  7. (k+l)α=kα+lα(k+l) \alpha = k\alpha + l \alpha(k+l)α=kα+lα
  8. k(α+β)=kα+kβk(\alpha + \beta) = k \alpha + k \betak(α+β)=kα+kβ
    其中α,β,γ\alpha,\beta,\gammaα,β,γ是VVV中的任意元素.

注: 即如果一个非空集合VVV在数域FFF上加法和数乘封闭，并且满足上述八条算律，则称VVV是数域FFF上的向量空间

子空间

定义2 \quad设VVV是一个FFF上的一个向量空间，那么VVV的一个子空间WWW满足:

  • WWW是VVV的一个非空子集.
  • WWW也是数域FFF上的一个向量空间.

只有一个零元素的集合叫做零空间.
一个向量空间VVV本身和零空间叫做VVV的平凡子空间. VVV的非平凡子空间叫做VVV的真子空间.

定理1 \quad 令WWW是数域FFF上向量空间VVV的一个非空子集. 如果WWW对于VVV的加法以及标量的乘法来说是封闭的,那么WWW是VVV的一个子空间.
证明: 由于WWW是VVV的一个非空子集，所以只需要证明WWW自身也作一个FFF上的向量空间即可.下面证明WWW也是一个向量空间.
首先WWW的加法的封闭以及标量的乘法性满足了向量空间的加法代数运算和数乘运算. 而对于算律，不难发现第1,2,5,6,7,8条算律都成立. 只需要证明第3和第4条算律即可.
因为WWW满足对数乘的封闭性,所以对于任意的α∈W\alpha \in Wα∈W，0α=0∈W0 \alpha = 0 \in W0α=0∈W,−α=(−1)α∈W-\alpha = (-1) \alpha \in W−α=(−1)α∈W，所以算律3,4也成立.

证明:

aα+bβ∈W1,aα+bβ∈W2a \alpha + b \beta \in W_{1}, \quad a \alpha + b \beta \in W_{2} aα+bβ∈W1​,aα+bβ∈W2​

向量的线性性质

线性相（无）关的定义

定义1：\quad 设α1,α2,⋯ ,αn\alpha_{1},\alpha_{2},\cdots,\alpha_{n}α1​,α2​,⋯,αn​是向量空间的nnn个向量，a1,an,⋯ ,ana_{1},a_{n},\cdots,a_{n}a1​,an​,⋯,an​是数域FFF中任意的nnn个数. 我们把

a1α1+a2α2+⋯+anαna_{1} \alpha_{1} + a_{2} \alpha_{2} + \cdots + a_{n}\alpha_{n} a1​α1​+a2​α2​+⋯+an​αn​

叫做向量α1,α2,⋯ ,αn\alpha_{1},\alpha_{2},\cdots,\alpha_{n}α1​,α2​,⋯,αn​的一个线性组合.

如果向量空间VVV中某一向量α\alphaα可以表示成向量α1,α2,⋯ ,αn\alpha_{1},\alpha_{2},\cdots,\alpha_{n}α1​,α2​,⋯,αn​的线性组合，我们也说α\alphaα可以由α1,α2,⋯ ,αn\alpha_{1},\alpha_{2},\cdots,\alpha_{n}α1​,α2​,⋯,αn​线性表示或者线性表出。

定义2: \quad 设α1,α2,⋯ ,αn\alpha_{1},\alpha_{2},\cdots,\alpha_{n}α1​,α2​,⋯,αn​是向量空间的nnn个向量. 如果存在FFF中不全为零的数a1,a2,⋯ ,ana_{1},a_{2},\cdots,a_{n}a1​,a2​,⋯,an​，使得

a1α1+a2α2+⋯+anαn=0(1)a_{1} \alpha_{1} + a_{2} \alpha_{2} + \cdots + a_{n}\alpha_{n} = 0 \tag{1} a1​α1​+a2​α2​+⋯+an​αn​=0(1)

那么就说α1,α2,⋯ ,αn\alpha_{1},\alpha_{2},\cdots,\alpha_{n}α1​,α2​,⋯,αn​线性相关.

反之，若(1)(1)(1)不存在不全为零的a1,a2,⋯ ,ana_{1},a_{2},\cdots,a_{n}a1​,a2​,⋯,an​使得(1)(1)(1)式成立，即(1)(1)(1)式成立当且仅当a1=a2=⋯=an=0a_{1} = a_{2} = \cdots = a_{n} = 0a1​=a2​=⋯=an​=0时，那么就说α1,α2,⋯ ,αn\alpha_{1},\alpha_{2},\cdots,\alpha_{n}α1​,α2​,⋯,αn​线性无关。

线性相（无）关的相关性质

现在我们直接从定义推导出一些简单的性质。

定理1: \quad 向量组{α1,α2,⋯ ,αn}\{\alpha_{1},\alpha_{2},\cdots,\alpha_{n}\}{α1​,α2​,⋯,αn​}中每一个向量αi\alpha_{i}αi​都可以由这一组向量线性表示.

证明:

γ=∑i=1raiβi=∑i=1rai∑j=1nbjαj=∑j=1n(∑i=1raibj)αj\begin{aligned} \gamma & = \sum_{i = 1}^{r} a_{i} \beta_{i} \\ & = \sum_{i = 1}^{r} a_{i} \sum_{j = 1}^{n} b_{j} \alpha_{j} \\ & = \sum_{j = 1}^{n} (\sum_{i = 1}^{r} a_{i}b_{j}) \alpha_{j} \end{aligned} γ​=i=1∑r​ai​βi​=i=1∑r​ai​j=1∑n​bj​αj​=j=1∑n​(i=1∑r​ai​bj​)αj​​

a1α1+a2α2+⋯amαm+0am+1+⋯+0an=0a_{1} \alpha_{1} + a_{2} \alpha_{2} + \cdots a_{m} \alpha_{m} + 0 a_{m+1} + \cdots + 0 a_{n} = 0 a1​α1​+a2​α2​+⋯am​αm​+0am+1​+⋯+0an​=0

定理4: \quad 设向量组{α1,α2,⋯ ,αn}\{\alpha_{1},\alpha_{2},\cdots,\alpha_{n}\}{α1​,α2​,⋯,αn​}线性无关，而{α1,α2,⋯ ,αn,β}\{\alpha_{1},\alpha_{2},\cdots,\alpha_{n},\beta\}{α1​,α2​,⋯,αn​,β}线性相关. 那么β\betaβ一定可以由α1,α2,⋯ ,αn\alpha_{1},\alpha_{2},\cdots,\alpha_{n}α1​,α2​,⋯,αn​线性表示.

替换定理

定义3: \quad 设{α1,α2,⋯ ,αn}\{\alpha_{1},\alpha_{2},\cdots,\alpha_{n}\}{α1​,α2​,⋯,αn​}和{β1,β2,⋯ ,βm}\{\beta_{1},\beta_{2},\cdots,\beta_{m}\}{β1​,β2​,⋯,βm​}是向量空间VVV的两个向量组. 如果每一个αi\alpha_{i}αi​都可以由β1,β2,⋯ ,βm\beta_{1},\beta_{2},\cdots,\beta_{m}β1​,β2​,⋯,βm​线性表示，而每一个βj\beta_{j}βj​也可以由α1,α2,⋯ ,αn\alpha_{1},\alpha_{2},\cdots,\alpha_{n}α1​,α2​,⋯,αn​线性表示，那么就说这两个向量组等价。

{α1,α2,⋯ ,αn}(2)\{\alpha_{1},\alpha_{2},\cdots,\alpha_{n}\} \tag{2} {α1​,α2​,⋯,αn​}(2)
