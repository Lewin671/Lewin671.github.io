分割、合并字符串

字符串对象的split()方法可以实现分割字符串，也就是把一个字符串按照指定的分隔符切分成字符列表。语法格式如下：
str.split(sep, maxsplit)
参数说明：

  • str: 表示要进行分割的字符串
  • sep： 用于指定分隔符，默认为None，即所有的空字符
  • maxsplit： 可选参数，用来指定分割的次数。如果不指定或者为-1，则分割次数没有限制。
  • 返回值： 分割后的字符串列表

检索字符串

count()方法

语法：

str.count(sub,[,start[,end]])

参数说明：

  • sub： 表示要检索的字串
  • start： 可选参数，表示检索的开始位置。默认为0.
  • end： 可选参数，表示检索范围的结束位置，如果不指定，则一直检索到结尾。

find()方法

用于检索字符串的起始位置。如果不存在则返回-1.语法格式如下：
str.find(sub[,start[,end]]
参数和count的含义一样。

index()方法

index和find方法基本一样，只是如果检索的字符串不存在，就会抛出异常。

startwith()方法/endwith()方法

用来判断字符串的起始（结束）是否为指定字符。语法格式如下：
startwith(str[,start[,end]])

去除字符串中的空格和特殊字符

使用strip()方法可以去除字符串中指定的字符。语法格式如下：
str.strip([char])
默认将空白字符去除。

字符串编码转换

使用encode()进行编码

语法：
str.encode([encoding = 'utf-8'][,error = "strict"]

  • errors = “strict”: 可选参数，用于指定错误处理的方式，默认为strict（遇到非法字符就抛出异常)
