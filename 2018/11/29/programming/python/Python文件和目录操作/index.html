Python文件和目录操作

Posted on 2018-11-29 Edited on 2020-04-27 In Programming Language , Python

基本文件操作

创建和打开文件

在Python中，想要操作文件需要先创建或者打开指定文件，可以通过open（）函数实现。open（）函数的语法格式如下：

file = open(filename[,mode[,buffering]])

其中[]里面的参数表示是可选参数。

  • filename 表示文件路径
  • mode表示文件打开的方式
  • buffering: 用于指定读写文件的缓冲模式。值为0表示不缓存，值为1表示缓存，如果大于1表示缓冲区的大小。

打开一个不存在的文件时先创建该文件

在默认情况下，使用open打开一个不存在的文件会抛出异常。解决的方法有两种：

  • 自己手动在该路径建立一个文件
  • 打开文件的的时候赋予写权限，比如：w,w+,a,a+等等。这样，当要打开一个不存在的文件时，就可以创建一个新的文件了。

打开文件指定编码方式

在使用open函数打开文件时，默认采用GBK编码，当被打开的文件不是GBK编码时，将会抛出异常，这个时候就需要我们指定打开文件的编码方式。
比如：

file = open('notice.txt','r',encoding='utf8')

关闭文件

打开文件之后，需要及时关闭，以免对文件造成不必要的破坏。close方法就时用来关闭文件的，但是它会先刷新缓冲区还没有写入的信息，然后再关闭文件。

写入文件内容

Python提供了write（）方法，可以向文件中写入内容。write方法的语法格式如下：

file.write(string)

其中，file是打开的文件对象，string时要写入的字符串。
注意：Python字符串可以是二进制数据，而不是仅仅是文字。

读取文件

  • 读取指定文件： file.read([size])，size是文件的字节数，如果省略size，则一次性读取所有内容
    在使用read(size)读取文件时，是从文件的开头开始读取的，如果想要读取部分内容，可以使用seek()方法将文件的指针移动到新的位置，然后再应用read方法读取。

    flie.seek(offset[,whence]): 文件指针移动到从whence开始计算，偏移量为offset的位置。如果省略whence，则默认为0

  • 读取一行： file.readline()

  • 读取全部行： file.readlines()

目录操作

在Python中，内置了os模块以其子模块os.path用于对目录或文件夹进行操作。在使用os模块或者os.path模块时，需要先应用import语句将其导入： import os

os中的常用变量

  • name: 用于获取操作系统类型。Windows下则表示nt,如果是Linux或者时Mac OS操作系统，则是posix。

  • linesep: 用于获取当前操作系统的换行符号。

  • sep： 用于获取操作系统所使用的路径分隔符号。

路径

  • 相对路径： 相对与当前目录的路径。当前工作目录可以通过os.getcmd()来获得。
  • 绝对路径： 绝对路径是在文件指定的实际路径，它不依赖当前目录。可以通过os.path.abspath()获得。
  • 拼接路径： 多个路径可以拼接成一个路径，这里不使用字符串拼接的原因是os.path.join()可以处理不同操作系统的路径分隔符。

判断目录是否存在

方法：

os.path.exists(path)

其中path是要判断的目录，可以采用相对路径，也可以采用绝对路径。
说明： os.path.exists()除了可以判断目录是否存在，还可以判断文件是否存在。

创建目录

如果是创建一级目录，则可以使用os.mkdir(path,mode=0o777).如果该目录的上一级目录不存在，则会抛出异常。
如果是创建多级目录，则使用os.makedirs(path,mode=0o777)

删除目录

如果删除的是空目录，则可以使用os.rmdir(path)。
否则需要使用Python内置的shutil的rmtree(path)函数实现。

遍历目录

在Python中，可以使用下面这个函数来遍历所有的目录和文件。

os.walk(top,[,topdown ][,oneerror] [,followlinks])

  • top: 用于指定要遍历的内容
  • topdown: 可选参数。用来指定遍历的顺序，如果为true，则表示自上而下，否则表示自下而上。
  • oneerror： 可选参数，表示指定错误的处理方式，默认为忽略，如果不想忽略也可以指定一个错误处理函数。通常采用默认设置。
  • onelinks： 可选参数。默认情况下，walk()函数不会向下转换成解析到目录的符号连接，将该参数值设置成True，用于指定支持系统上访问由符号链接指向的目录。
  • 返回值： 返回一个包含三个元素(dirpath,dirnames,filenames)的元组生成器对象。
    但是很遗憾，walk函数只能在Unix系统和windows系统下使用。

获取文件的基本信息

通过os模块的stat()函数可以获取文件的基本信息。语法格式如下：

os.stat(path)

返回的是一个对象，包含以下属性：

  • st_mode: 保护模式
  • st_ino: 索引号
  • st_nlink: 硬链接号(被连接数目)
  • st_size: 文件大小，单位时字节
  • st_mtime: 最后一次修改的时间
  • st_dev: 设备名
  • st_uid: 用户id
  • std_gid: 组id
  • st_atime: 最后一次访问的时间
  • st_ctime: 最后一次状态变化的时间
