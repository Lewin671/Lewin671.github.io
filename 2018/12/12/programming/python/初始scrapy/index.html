安装scrapy模块

安装scrapy最好参照官方文档，不然可能会有很多坑。

  1. To install scrapy on Ubuntu (or Ubuntu-based) systems, you need to install these dependencies:
    sudo apt-get install python-dev python-pip libxml2-dev libxslt1-dev zlib1g-dev libffi-dev libssl-dev
  2. If you want to install scrapy on Python 3, you’ll also need Python 3 development headers:
    sudo apt-get install python3 python3-dev
  3. sudo pip3 install scrapy，如果是python2的scrapy，则sudo pip install scrapy

这些安装python2的scrapy是没有问题的，但是python3的scrapy却会出现有很多模块不存在的情况，这是因为twisted的版本不对或者没有安装，经过大量google，终于找到了合适的版本的twisted：
