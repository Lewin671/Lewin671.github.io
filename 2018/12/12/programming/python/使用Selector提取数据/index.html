使用Selector提取数据

Posted on 2018-12-12 Edited on 2020-04-27 In Programming Language , Python

Introduction

Scrapy有它自己的工具提取数据，这就是Selector。我们可以通过使用XPath和CSS表达式来提取数据。

Scrapy的Selector是根据lxml库编写的，所以它们的速度和解析的准确性都是非常接近的。

For a complete reference of the selectors API see Selector reference

Selector objects

构造函数

class scrapy.selector.Selector(response=None, text=None, type=None)
参数说明：

  • response: response是HtmlResponse或者XmlResponse对象,是要被抽取数据的数据源。

  • text: text是一个unicode或者uft8字符当一个response对象不可用的时候的文字.

  • type: type定义选择器的类型，有html、xml、None(default)。
    如果 type 是 None ，选择器会根据 response 类型(参见下面)自动选择最佳的类型，或者在和 text 一起使用时，默认为 “html” 。

    如果 type 是 None ，并传递了一个 response ，选择器类型将从response类型中推导如下：

所以我们既可以根据text来构建一个Selector对象，也可以根据response来构建一个Selector对象.

xpath(query)

寻找可以匹配xpath query 的节点，并返回 SelectorList 的一个实例结果，单一化其所有元素。列表元素也实现了 Selector 的接口。

为了方便起见，该方法也可以通过 response.xpath() 调用

css(query)

应用给定的CSS选择器，返回 SelectorList 的一个实例。

query 是一个包含CSS选择器的字符串。

在后台，通过 cssselect 库和运行 .xpath() 方法，CSS查询会被转换为XPath查询。

为了方便起见，该方法也可以通过 response.css() 调用

extract()

串行化并将匹配到的节点返回一个unicode字符串列表。 结尾是编码内容的百分比。

re(regex)

应用给定的regex，并返回匹配到的unicode字符串列表。、

regex 可以是一个已编译的正则表达式，也可以是一个将被 re.compile(regex) 编译为正则表达式的字符串

XPath

XPath即XML路径语言(XML Path Language)，它是一种用来确定xml文档中的某个部分位置的语言。

基础语法

Xpath的常用基本语法
表达式|描述 ---|---- /|选中文档的根 .|选中当前结点 ..|选中当前结点的父亲结点 ELEMENT|选中**子节点中**的所有ELEMENT元素的结点 //ELEMENT|选中后代节点中的所有ELEMENT元素的结点 *|选中所有元素子节点 text()|选中所有文本结点 @ATTR|选中名为ATTR的属性结点 @*|选中所有属性结点 [谓词] | 谓词用来查找某个特定的结点或者包含某个特定值的结点

注:
子节点是相对于父节点的下一层，只有一层，后代则是父节点下的所有元素

CSS選擇器

語法

表達式 描述
* 選中所有元素
E 選中E元素
E1，E2 選中E1和E2元素
E1 E2 選中E1後代中的E2元素
E1>E2 選中E1子元素中的E2元素
E1+E2 選中E1兄弟中的E2元素
.CLASS 選中CLASS屬性包含CLASS的元素
#ID 選中id屬性爲IDD元素
[ATTR] 選中包含ATTR屬性的元素
[ATTR=VALUE] 選中屬性爲ATTR且值爲VALUE的元素
[ATTR~=VALUE] 選中包含ATTR屬性且值包含VALUE的元素
E:nth-child(n)
E:nth-last-child(n)		 選中E元素，且該元素必須是其父親元素的（倒數）第n個子元素
E:first-child
E:last-child		 選中E元素，且改元素必須是其父元素的（倒數）第1個元素
E:empty 選中沒有子元素的E元素
E::text 選中E元素的文本節點(TEXT NODE)
