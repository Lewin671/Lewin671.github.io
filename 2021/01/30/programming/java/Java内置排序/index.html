Java内置排序

Posted on 2021-01-30 In Programming Language , Java

在Java的内置排序算法中，一般有两种静态排序的方法。

  • Arrays.sort
  • Collections.sort

其中Arrays.sort方法也有两种不同的算法:

  • 针对原始类型数组，使用快速排序算法DualPivotQuicksort
  • 针对对象类型数组，使用稳定的TimSort算法或者是LegacyMergeSort

对于对象类型数组的排序，可以传入一个Comparator对象进行设定结果的顺序。
